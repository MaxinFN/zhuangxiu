## 木工验收

木工验收的核心是**轻钢龙骨吊顶**（结构安全+防开裂）和**现场木作**（尺寸精度+收口工艺）。吊顶的"L型整板转角""螺丝防锈"是验收中容易遗漏但对长期使用影响最大的两项。

---

## 一、吊顶验收

**依据**：GB 50210-2018《建筑装饰装修工程质量验收标准》、GB/T 11981-2008《建筑用轻钢龙骨》

### 1.1 龙骨验收（封板前检查）

封石膏板前必须检查龙骨系统（封板后龙骨被遮盖，无法再验）：

- [ ] 吊杆垂直、间距≤900mm（主龙骨方向）/ ≤1200mm，距墙≤300mm有第一道吊杆
- [ ] 吊杆用膨胀螺栓固定于混凝土顶面（非自攻钉），入混凝土深度≥30mm
- [ ] 边龙骨用膨胀螺栓固定于墙面，间距≤400mm
- [ ] 主龙骨间距≤900mm，副龙骨间距400mm（9.5mm板）/ 300mm
- [ ] 所有龙骨连接用专用挂件，无铁丝绑扎、无木楔垫撑
- [ ] 龙骨调平：所有副龙骨底面在同一水平面（激光水平仪检查偏差≤2mm）
- [ ] 吊顶内管线已完成且不影响龙骨
- [ ] 灯具加固件已预装（吊灯位置用OSB板或轻钢横撑加固、浴霸位用加强龙骨）

### 1.2 石膏板封板验收

| 检查项 | 合格标准 | 检查方法 |
|--------|---------|---------|
| **转角L型整板** | 吊顶所有转角处石膏板为一块整板裁成L型（非两块板在转角拼接） | 目视检查每个转角 |
| 板缝 | 板与板之间留3-5mm均匀伸缩缝 | 目视+钢尺 |
| 双层板错缝 | 上下两层石膏板接缝错开≥300mm（如有双层板封板） | 目视 |
| 自攻螺丝间距（板边） | ≤150mm | 钢尺抽查 |
| 自攻螺丝间距（板中） | ≤200mm | 钢尺抽查 |
| 自攻螺丝深度 | 螺丝头陷入板面0.5-1mm（以纸面未破为限） | 手摸无凸出感 |
| **螺丝防锈** | **所有螺丝头已点涂防锈漆** | 逐颗目视（不能漏！） |
| 石膏板无破损、无裂缝 | 目视 | — |

### 1.3 接缝处理验收

- [ ] 板缝已填入嵌缝石膏，压实无空隙
- [ ] 接缝纸带已嵌入并覆盖嵌缝膏（非玻纤网格带——纸带抗裂性能更好）
- [ ] 嵌缝膏宽度≥150mm（纸带每侧覆盖≥75mm）
- [ ] 表面已打磨平整、无凸出

### 1.4 吊顶水平度

| 项目 | 合格标准 | 工具 |
|------|---------|------|
| 吊顶标高 | 与设计图一致 | 激光水平仪+卷尺 |
| 表面平整度 | ≤2mm/2m | 2m靠尺+塞尺 |
| 接缝高低差 | ≤0.5mm | 钢尺+塞尺 |

## 二、窗帘盒验收

- [ ] 宽度：单层帘≥150mm / 双层帘（布+纱）≥220mm / 电动双轨≥260mm
- [ ] 深度（顶面到石膏板底）≥100mm
- [ ] 电动窗帘预留电源线已到位（86底盒+220V，位于窗帘盒内任意一端）
- [ ] 窗帘盒使用细木工板/OSB板基层（非仅石膏板——无握钉力后期窗帘轨道安装会拉垮）

---

## 三、现场柜体验收（如有）

### 3.1 尺寸与安装

- [ ] 柜体水平度：水平尺放在柜体顶板上检查，≤2mm
- [ ] 柜体垂直度：水平尺贴柜体侧板，≤2mm/1m
- [ ] 柜体与墙体固定牢固（膨胀螺栓+连接件，非仅打胶或自攻钉）
- [ ] 柜体底部可调脚已调平（非垫纸片/木楔）
- [ ] 柜门缝隙均匀（偏差≤1mm），每扇门开关3次以上顺畅无卡阻
- [ ] 抽屉全拉出顺畅（三节轨全拉出测试），阻尼缓冲自然
- [ ] 拉手安装在同一水平线上（水平尺检查）

### 3.2 板材与封边

- [ ] 板材品牌与合同一致，有防伪标识可查
- [ ] 板材环保等级标识清晰（E0/ENF标识在板材侧面喷码）
- [ ] 现场手工封边（如有）无翘边、无脱胶（但需理解：手工封边效果不及工厂PUR封边）

---

## 四、集成吊顶验收（厨卫）

- [ ] 龙骨为轻钢龙骨（非木龙骨）——厨卫潮湿，木龙骨会变形腐烂
- [ ] 铝扣板安装平整、无下垂
- [ ] 板与板之间的缝隙均匀一致（≤0.5mm）
- [ ] 灯具/浴霸/凉霸安装牢固、开孔位置正确
- [ ] 检修口预留（尺寸≥200×200mm）且可正常开合
- [ ] 浴霸排气管道通向室外或公共排风道（非排入吊顶内）

## 常见验收问题

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 吊顶转角为两块板拼接（非L型整板） | 🔴 严重 | 拆掉重做——此处必裂 |
| 螺丝头未点防锈漆 | 🟡 中等 | 逐颗补点防锈漆 |
| 吊杆使用铁丝绑扎（非专用挂件） | 🔴 严重 | 更换为专用挂件 |
| 石膏板有明显裂缝或破损 | 🟡 中等 | 拆除破损板重新封板→重新接缝处理 |
| 窗帘盒宽度/深度不足（装不了双层帘） | 🔴 严重 | 拆除重做 |
| 吊顶水平度偏差>5mm | 🟡 中等 | 拆板→重新调平龙骨→重新封板 |
| 集成吊顶用木龙骨 | 🔴 严重 | 拆除→更换为轻钢龙骨 |
| 浴霸排气管排入吊顶内 | 🔴 严重 | 接硬质管道至室外或公共排风道 |
