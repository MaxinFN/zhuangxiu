## 木工施工工艺

当前主流做法是"吊顶现场做+柜体工厂定制"。现场木工主要集中在轻钢龙骨吊顶、造型背景墙和局部木作基层。以下按工序详述。

---

## 一、轻钢龙骨吊顶

### 1.1 施工流程

```
弹线定位 → 安装吊杆 → 安装边龙骨 → 安装主龙骨 → 安装副龙骨 → 
龙骨调平 → 封石膏板 → 接缝处理 → 螺丝点防锈漆 → 验收
```

### 1.2 弹线定位

| 步骤 | 工艺要求 | 工具 |
|------|---------|------|
| 标高线 | 用激光水平仪放出吊顶完成面标高，沿四周墙面弹一圈闭合的水平线 | 激光水平仪+墨斗 |
| 龙骨线 | 在地面弹出主龙骨和副龙骨的投影线（用于定位吊杆和龙骨位置） | 墨斗 |
| 灯具点位 | 在顶面标出所有筒灯/射灯/吊灯的准确位置（与灯位图一致） | 记号笔 |

> 标高确定原则：下吊高度 = 管线最低点高度 + 龙骨+石膏板厚度（约50mm）+ 灯具安装空间（筒灯需≥80mm）

### 1.3 吊杆安装

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 吊杆直径 | ≥Φ6mm（常规）/ ≥Φ8mm（吊顶面积>50㎡或上人吊顶） | 材质：Q235热镀锌 |
| 吊杆间距 | ≤900mm（主龙骨方向）/ ≤1200mm | 距墙≤300mm（第一根吊杆） |
| 固定方式 | M8膨胀螺栓（混凝土顶面）或自攻钉+膨胀管（木基层） | 入混凝土深度≥30mm |
| 垂直度 | 吊杆必须垂直（偏斜≤3°），禁止斜拉 | 斜拉吊杆会受力不均 |

### 1.4 龙骨安装

| 龙骨 | 规格 | 间距 | 连接方式 |
|------|------|------|---------|
| 边龙骨（沿墙L型） | 20×30×0.5mm | 膨胀螺栓固定，间距≤400mm | M6膨胀螺栓 |
| 主龙骨（承载龙骨） | DU38×12×1.0mm 或 DU50 | ≤900mm | 主挂件连接吊杆 |
| 副龙骨（覆面龙骨） | DC50×19×0.5mm 或 DC60 | 400mm（9.5mm板）/ 300mm（12mm板或重载区） | 副挂件卡入主龙骨 |
| 贯穿龙骨 | 38×12mm | 每≤800mm一道 | 穿入副龙骨专用孔 |

> **龙骨调平**：全部龙骨安装完成后用激光水平仪校准，调节吊杆螺帽使所有副龙骨底面在同一水平面上（偏差≤2mm）。

### 1.5 封石膏板

| 参数 | 标准做法 | 原因 |
|------|---------|------|
| 铺板方向 | 石膏板长边**垂直于**副龙骨方向 | 增加板面刚度 |
| 板缝 | 板与板之间留3-5mm伸缩缝 | 防止温湿度变化导致挤压起拱 |
| 转角处理 | **必须用整板裁成L型**，不能用两块板在转角拼缝 | 转角是应力集中点，拼缝必裂 |
| 双层板 | 上下两层板缝必须错开≥300mm | 防止贯通裂缝 |
| 自攻螺丝间距 | 板边≤150mm，板中≤200mm | 螺钉距板边≥10mm、距板角≥15mm |
| 自攻螺丝深度 | 螺丝头陷入板面0.5-1mm（以不破坏纸面为限） | 凸出则抹灰不平，太深则失去握裹力 |
| 螺丝防锈 | 每个螺丝头点涂防锈漆（黑色磷化自攻螺丝的防锈层在拧入时可能磨损） | 不点漆→螺丝生锈→锈斑透过漆面 |

### 1.6 接缝处理

| 步骤 | 工艺 |
|------|------|
| 1. 切V型槽 | 石膏板拼缝处用刀切V型槽（深约板厚1/3） |
| 2. 嵌缝膏填充 | 用嵌缝石膏填满板缝，压实（一次不要填太厚） |
| 3. 贴接缝纸带 | 在嵌缝膏未干时将纸带嵌入压实，用刮刀刮平挤出多余膏体 |
| 4. 覆盖嵌缝膏 | 纸带上再刮一层嵌缝膏，宽度≥150mm（每边≥75mm） |
| 5. 干燥打磨 | 完全干燥后用240#砂纸打磨平整 |

> **接缝纸带 vs 玻纤网格带**：纸带抗拉强度远高于网格带（干后≈1000N/50mm vs≈200N/50mm），优选纸带。

## 二、造型背景墙

### 2.1 石膏板背景墙

- 先用OSB板（≥9mm）作基层衬板→再覆石膏板（双层结构）
- 基层OSB板用膨胀螺栓固定于原墙
- 需要悬挂电视的区域，必须在OSB基层内预埋≥18mm多层板加强块（尺寸≥400×400mm），确保电视挂架螺栓能吃在加强块上

### 2.2 木饰面背景墙

- 基层：OSB板或多层板（≥12mm），需调平（木楔或可调基层件）
- 面层：实木贴皮面板（≥3mm厚木皮）/ 科定板（KD板）
- 收口：阳角45°拼角、阴角留工艺缝（3-5mm）
- 粘贴：推荐使用MS免钉胶或结构胶（非溶剂型），均匀点状或条状施加

## 三、窗帘盒

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 宽度（单层帘） | ≥150mm | 含轨道+窗帘褶皱空间 |
| 宽度（双层帘，布+纱） | ≥220mm（手动）/ ≥260mm（电动双轨） | 双轨间距≥50mm |
| 深度 | ≥100mm（从石膏板底到顶面） | 太浅帘头外露 |
| 电动窗帘 | 预留220V电源（距顶面≤200mm） | 用86型底盒预埋，位置在窗帘盒内任意一端 |
| 结构 | 细木工板/OSB板基层+石膏板 | 禁止仅用石膏板做窗帘盒（无握钉力） |

## 四、现场柜体制作

### 4.1 裁板要求

- 必须用导轨锯（精准度±1mm），不能用手持圆锯+靠尺
- 裁切面用PUR/EVA封边条手工封边（**现场封边无法达到工厂水平**，这是现场木作的最大软肋）

### 4.2 连接方式

| 连接件 | 适用 | 说明 |
|--------|------|------|
| 三合一连接件（偏心轮+螺杆+预埋螺母） | 柜体主体连接 | 标准工厂级连接（现场需精准打孔） |
| 二合一（层板托+销钉） | 活动层板 | 层板可上下调节 |
| 角码+L型连接片 | 加固、非主要受力 | 辅助加固 |
| 自攻螺丝 | 背板固定 | 背板≥5mm，嵌入槽内后用螺丝固定 |

### 4.3 水平调整

- 柜体就位后用激光水平仪检查前后、左右、顶面水平
- 底部用可调脚（±15mm调节范围）调水平
- 柜体与墙面缝隙用收口条遮盖（非打玻璃胶）

## 五、集成吊顶（厨卫铝扣板）

### 5.1 龙骨系统

| 部件 | 材质/规格 | 说明 |
|------|---------|------|
| 主龙骨 | 轻钢龙骨（镀锌），**禁止用木龙骨** | 厨卫潮湿，木龙骨变形腐烂 |
| 三角龙骨 | 镀锌钢，间距按铝扣板规格 | 300×300板间距300mm/600×600板间距600mm |
| 吊杆 | Φ6mm镀锌全丝杆 | 间距≤1200mm |

### 5.2 安装顺序

1. 放出集成吊顶标高线（浴霸/凉霸位置提前加固龙骨）
2. 安装吊杆+主龙骨+调水平
3. 安装三角龙骨
4. 从一侧向另一侧依次卡入铝扣板
5. 安装灯具/浴霸/凉霸（开孔在安装扣板时同步进行）
6. 最后一块板裁切到合适尺寸卡入（通常靠墙侧）

### 5.3 检修口

- 厨房烟道检修口（如有）和卫生间排水管检查口上方必须预留活动检修口
- 最小检修口尺寸≥200×200mm
