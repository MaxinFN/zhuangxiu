## 木工施工核心工艺

### 一、轻钢龙骨吊顶

**标准施工流程：**

```
弹线定位 → 安装吊杆 → 安装主龙骨 → 安装副龙骨 → 
封石膏板 → 接缝处理 → 点防锈漆
```

#### 1. 弹线定位

- 用激光水平仪放出吊顶完成面标高（通常下吊100-300mm）
- 沿墙弹线，标记边龙骨位置

#### 2. 吊杆安装

- 间距：主龙骨方向≤900mm，副龙骨方向≤1200mm
- 用膨胀螺栓固定于混凝土顶面
- 吊杆必须**垂直**，不能斜拉

#### 3. 龙骨安装

- 主龙骨间距≤900mm，用挂件连接吊杆
- 副龙骨间距：400mm（9.5mm石膏板）或300mm（造型复杂处）
- 龙骨连接用专用连接件，不能用铁丝绑

#### 4. 封石膏板

**要点：**
- 石膏板**长边垂直**副龙骨方向（增加强度）
- 板与板之间留3-5mm缝隙（用于嵌缝膏填充）
- 转角处石膏板必须用**整板裁成L型**（不能用两块板拼接）
- 自攻螺丝间距：板边150mm，板中200mm
- 螺丝头陷入板面0.5-1mm（不能凸出，也不能打穿）

**为什么转角要用L型整板？**

> 吊顶转角处受应力集中，如果两块板在这里拼接，一定会开裂。用L型整板可以分散应力，大大减少开裂概率。

#### 5. 接缝处理

```
嵌缝膏填缝 → 贴接缝纸带 → 刮平 → 干燥 → 打磨
```

- 接缝纸带比网格带好（纸带强度更高）
- 螺丝头必须点防锈漆（自攻螺丝的防锈层在拧入时可能破损）

### 二、窗帘盒制作

- 宽度：单层窗帘≥150mm，双层窗帘≥250mm
- 深度：顶到石膏板下沿≥100mm
- 如果计划装电动窗帘，预留电源线

### 三、现场柜体制作

**关键工艺点：**

1. **裁板**：用导轨锯（不能手锯），保证切口平直
2. **封边**：这是现场木作最大的短板——手工封边无法和工厂PUR封边比
3. **组装**：用三合一连接件或角码，不能用钉子硬钉
4. **背板**：至少5mm厚，需开槽嵌入
5. **调水平**：柜体就位后用水平仪调平，底部垫调整脚

### 四、集成吊顶（厨卫）

- 用轻钢龙骨（不能用木龙骨——厨卫潮湿，木龙骨会变形腐烂）
- 铝扣板安装从一侧向另一侧推进
- 浴霸/凉霸位置提前加固龙骨
- 预留检修口（Φ200mm以上）
