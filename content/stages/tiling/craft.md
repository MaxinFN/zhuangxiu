## 瓦工施工工艺

瓦工是装修中涉及工序最多、技术要求最综合的阶段——防水、贴砖、石材铺贴、地漏安装，每一项都直接决定日后的使用体验和邻里关系。以下按 **GB 50209-2010（地面）** 和 **JGJ 298-2013（防水）** 的要求详述每个工艺。

---

## 一、防水施工

### 1.1 整体流程

```
基层清理 → 节点加强（管根/墙角/门槛石） → 第一遍防水涂刷 → 干燥(4-8h) →
第二遍防水涂刷（与第一遍方向交叉） → 干燥养护24-48h → 闭水试验≥48h → 防水保护层
```

### 1.2 基层处理

| 步骤 | 工艺要求 | 检验方法 |
|------|---------|---------|
| 清扫 | 无浮灰、无油污、无突出物 | 手摸无尘 |
| 管根处理 | 管周凿V形槽（宽20mm×深15mm），用堵漏王（速凝型水不漏）填实抹平 | 目视 |
| 阴阳角处理 | 做成圆弧倒角（R≥20mm），或用堵漏王做45°倒角 | 直角处涂料易开裂 |
| 地漏根部 | 地漏周边V形槽+堵漏王封堵（同管根） | — |
| 门槛石位置 | 门槛石下方先行浇筑混凝土挡水坎（高≥30mm，宽同墙厚），防水层翻过挡水坎 | 这是卫生间防水的第一道防线 |

### 1.3 防水涂刷

**执行标准**：GB/T 23445-2009《聚合物水泥防水涂料》、JGJ 298-2013《住宅室内防水工程技术规范》

| 涂刷参数 | 标准值 | 说明 |
|---------|--------|------|
| 涂刷遍数 | ≥2遍（推荐3遍：底涂+面涂+加强涂） | 两遍交叉方向（第一遍横向，第二遍竖向） |
| 每遍厚度 | 0.5-0.8mm（湿膜） | 干膜总厚≥1.2mm |
| 每遍间隔 | 指触干（约4-8h，视温湿度） | 未干透就刷第二遍会起泡、结合不牢 |
| 涂刷高度（淋浴区墙面） | ≥1800mm | 淋浴水花飞溅范围 |
| 涂刷高度（卫生间其他墙面） | ≥300mm | 拖地溅水范围 |
| 涂刷高度（浴缸墙面） | ≥浴缸上沿+300mm | — |
| 管根/墙角 | 加无纺布增强层（宽度≥200mm，嵌入涂层中间） | 最关键的防漏细节 |

| 品牌 | 适用产品 | 施工特性 |
|------|---------|---------|
| 雨虹 | 好仕涂100（JS II型）/蓝色金典G101 | 开桶即用，粉料+液料混合搅拌3-5min |
| 德高 | K11通用型/柔韧型 | 通用型适合墙面，柔韧型适合地面 |
| 西卡 | SikaTop Seal-107 | 双组份，粉料+乳液 |

### 1.4 防水涂料用量

- JS II型防水涂料（双组份）：1.5-2.0 kg/㎡/遍（涂刷2-3遍时总用量约4-6 kg/㎡）
- 18kg桶装可涂刷约4-6㎡（3遍总涂刷面积）
- 参考：一个4㎡卫生间（地面4㎡+墙面约15㎡）约需3-4桶

### 1.5 闭水试验

**执行标准**：GB 50209-2010《建筑地面工程施工质量验收规范》第4.10.13条。

| 参数 | 要求 |
|------|------|
| 蓄水深度 | ≥20mm（卫生间门口用沙袋或砖+水泥临时筑坝挡水） |
| 蓄水时间 | ≥24h（推荐48h，更保险） |
| 检查方法 | ① 观察水位线是否下降（用记号笔记下初始水位）；② 到楼下邻居家对应位置用手电检查天花板——看有无湿痕、渗水印、滴水 |
| 合格标准 | 水位无明显下降（蒸发损失可忽略不计），楼下天花板无任何渗漏痕迹 |
| 记录 | 拍照+物业或业主签字确认**

> **必须提前通知物业和楼下邻居**。进入楼下邻居家检查时注意保护对方家装（脱鞋或穿鞋套）。

## 二、瓷砖铺贴

### 2.1 预排版（铺贴前必做）

1. 在CAD或手绘图纸上按实际瓷砖规格排列，确定**起铺点**
2. 起铺点一般选在：入户门中线、客厅中心、或走廊中轴线
3. 预排后检查：有无窄条（裁切宽度<原砖宽度1/3的尽量调整排布避免）、走廊与客厅的缝是否通缝对齐、墙砖与地砖缝是否对缝
4. 现场**干铺预排**：在铺贴前将砖按排版图直接放在地面试排，确认无误后再正式铺贴

### 2.2 铺贴方法对比

| 方法 | 粘结材料 | 粘结层厚 | 粘结强度（28d标养） | 适用 |
|------|---------|---------|-------------------|------|
| 薄贴法（瓷砖胶） | C1/C2水泥基瓷砖胶 | 3-6mm | ≥0.5MPa（C1）/ ≥1.0MPa（C2） | 墙砖、大板（≥600×1200）、低吸水率砖（<0.5%） |
| 厚贴法（水泥砂浆） | 1:3水泥砂浆（32.5级） | 20-40mm | ≥0.3MPa | 地砖（普通规格）、水泥基层较平整 |
| 半干湿铺法 | 干硬性水泥砂浆（手握成团落地即散）+ 素水泥浆 | 25-50mm | ≥0.3MPa | 大地砖、石材地面 |

### 2.3 薄贴法工艺要点（墙砖）

| 步骤 | 工艺要求 |
|------|---------|
| 基层找平 | 墙面平整度≤3mm/2m（不达标须先找平） |
| 瓷砖背面处理 | 低吸水率砖（<0.5%）须刷涂乳液型背胶，干燥至透明（约30-60min） |
| 瓷砖胶搅拌 | 按说明书加水，电动搅拌至无颗粒，静置5min后再搅拌一次 |
| 刮涂 | 用齿形刮刀在墙面刮瓷砖胶（齿形方向一致），瓷砖背面也薄刮一层（双面刮涂） |
| 粘贴 | 将砖按压到位，用橡皮锤轻敲调整水平，十字卡控制缝隙 |
| 调平 | 用调平器（楔子+底座）夹在两片砖面，旋转楔子使两砖面平齐 |
| 养护 | 24h后拆除调平器和十字卡，48h内避免震动、踩踏（填缝一般在铺贴完成后7天） |

### 2.4 铺贴质量标准

| 项目 | 允许偏差 | 检验方法 |
|------|---------|---------|
| 表面平整度 | ≤2mm/2m（瓷砖） | 2m靠尺+塞尺 |
| 缝格平直度 | ≤2mm/5m | 拉5m线+钢尺 |
| 接缝高低差 | ≤0.5mm | 钢尺+塞尺 |
| 相邻砖高差 | ≤0.5mm | — |
| 空鼓率 | 单块砖≤15%，整面墙≤5% | 空鼓锤逐块敲击 |
| 卫生间地面坡度 | 向地漏方向≥1%（每米落差≥10mm） | 泼水测试/乒乓球滚动方向 |
| 阴阳角方正度 | ≤3mm | 直角尺+塞尺 |

### 2.5 阳角处理

| 方式 | 工艺 | 美观度 | 耐久 | 费用 |
|------|------|--------|------|------|
| **海棠角** | 两片砖各背切45°（留1-2mm釉面不切透），对拼成90°，留2mm缝 | 最美观 | ★★★★★ | ¥10-20/m |
| 阳角条 | PVC/铝合金/不锈钢条嵌入两砖之间 | 一般 | ★★★★ | ¥5-10/根 |
| 直接碰角 | 两片砖直接90°边缘相碰（未磨边） | 差（边缘锋锐割手） | ★★★ | — |

> 推荐海棠角。切砖在工厂或现场用切割机加工，留1-2mm"釉面安全边"（避免边缘过薄崩瓷）。

### 2.6 地漏安装

- 地漏芯体必须位于该区域**绝对最低点**
- 地漏四周瓷砖做**回字形切割**：沿地漏外框对角线切4刀，形成向地漏汇水的漏斗状斜面
- 排水管接入地漏前检查立管是否有水泥块堵塞（管道内窥镜或强光手电）
- 地漏芯选用：深水封型（水封高度≥50mm，符合 GB 50015-2019）或重力翻板+磁吸型
- 洗衣机地漏必须选用**洗衣机专用地漏**（带接头），不能用普通地漏替代

## 三、石材铺贴

### 3.1 过门石（门槛石）

- 卫生间门槛石**必须高于卫生间完成地面≥5mm**（挡水功能）
- 门槛石下方防水层必须与两侧防水层连为一体（形成U型防水兜底）
- 建议材质：天然大理石（黑金花/黑白根）或花岗岩（耐水耐磨）

### 3.2 窗台石

- 窗台石两端入墙≥20mm（嵌入墙体更稳固）
- 外侧下沿做"滴水线"或"倒角"（防止雨水沿窗台面倒流进墙体）
- 窗台石出墙面约10-15mm（保护窗台下墙面）

## 四、环氧彩砂/美缝施工

| 步骤 | 要点 |
|------|------|
| 清缝 | 用清缝刀将砖缝内杂物清理干净（深度≥3mm），吸尘器吸净 |
| 干燥 | 确保砖缝完全干燥（有水分会导致美缝剂不固化或发霉） |
| 打胶/填砂 | 环氧彩砂用橡胶刮板压填（双组份混合后操作时间约30-60min） |
| 压缝 | 用压缝球沿缝压实、刮平 |
| 清理余料 | 固化前（4-6h）用湿海绵打圈擦洗表面余料 |
| 养护 | 完全固化≥24h（期间防水、防尘） |
