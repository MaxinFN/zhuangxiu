## 瓦工验收

瓦工验收分三大块：防水验收（闭水试验是唯一标准）、瓷砖铺贴验收（空鼓+平整度+坡度）和石材辅材验收。**闭水试验业主必须亲自到楼下邻居家检查，不能只靠施工方口头报告。**

---

## 一、防水验收——闭水试验

**依据**：GB 50209-2010《建筑地面工程施工质量验收规范》第4.10.13条

| 参数 | 标准 | 验证方法 |
|------|------|---------|
| 蓄水深度 | ≥20mm | 卷尺测量水位距门槛石顶面的深度 |
| 蓄水时间 | **≥48h（推荐）** / ≥24h（最低） | 记录开始时间和水位标记 |
| 水位下降 | 无明显下降（蒸发损失可忽略） | 用记号笔在门槛石或墙面标记初始水位线 |
| 楼下天花板 | 无任何渗漏痕迹 | 到楼下邻居家对应位置用手电筒逐一检查天花板各处（尤其墙角、管根对应的顶面） |
| 墙角/管根 | 全部干燥 | 手摸+目视（管根是最容易漏水的位置） |

### 闭水试验的标准操作流程

1. 用堵头或沙袋封堵所有排水口
2. 注水至门槛石顶面以下10mm左右（水深≥20mm）
3. 用记号笔记下初始水位
4. 满48h后检查：
   - a) 水位是否明显下降（排除蒸发）
   - b) **到楼下邻居家**，用手电筒检查对应区域天花板
5. 试验结果拍照记录，业主和施工方签字确认

> ⚠️ **试验前必须通知物业和楼下邻居并取得配合**。如果楼下已装修完成，进入时脱鞋或穿鞋套，注意不要触碰对方家具。

## 二、瓷砖铺贴验收

**依据**：GB 50209-2010

### 2.1 空鼓检查（最重要）

| 区域 | 合格标准 | 检查方法 |
|------|---------|---------|
| 地砖 | 单块砖空鼓≤15%，整面地面≤5% | 空鼓锤逐块敲击（在砖中心和四角分别敲），声音清脆实心=合格，声音空洞=空鼓 |
| 墙砖 | 单块砖空鼓≤15%，整面墙≤5% | 同上 |
| 地砖中心区域 | **不得有空鼓** | 中心区域是日常踩踏最频繁的位置 |
| 过门石/门槛石 | 不得有空鼓 | — |

### 2.2 平整度与缝格

| 检查项 | 工具 | 合格标准 |
|--------|------|---------|
| 表面平整度（地砖） | 2m靠尺+塞尺 | ≤2mm/2m |
| 表面平整度（墙砖） | 2m靠尺+塞尺 | ≤2mm/2m |
| 缝格平直度 | 拉5m线+钢尺 | ≤2mm/5m |
| 接缝高低差 | 钢尺+塞尺 | ≤0.5mm |
| 相邻砖四角平整度 | 手摸+目视 | 手感平滑、无缝感 |
| 阴阳角方正度 | 直角尺+塞尺 | ≤3mm |

### 2.3 排水坡度

| 区域 | 标准 | 检查方法 |
|------|------|---------|
| 卫生间地面 | 向地漏方向≥1%（每米落差≥10mm） | 泼水测试：向地面泼水→观察水流方向→应全部流向地漏→无局部积水 |
| 阳台地面 | 向地漏方向≥1% | 同上 |
| 淋浴区 | 向地漏方向≥1.5% | 同上，坡度偏大更利于快速排水 |

### 2.4 其他检查

- [ ] 墙砖与地砖缝隙对齐（墙地对缝，如有此项要求）
- [ ] 阳角处理工艺到位：海棠角拼角均匀（缝宽一致，2mm左右）、无崩瓷
- [ ] 地漏回字形切割完整，四周坡度向地漏中心汇水
- [ ] 过门石安装牢固、无空鼓，卫生间门槛石高于卫生间地面≥5mm
- [ ] 瓷砖表面无划痕、无崩边、无色差（同一区域色号一致）
- [ ] 十字卡和调平器已全部拆除（铺贴完成≥24h后）

## 三、环氧彩砂/美缝验收

- [ ] 砖缝填充饱满、无漏填
- [ ] 表面平滑（手摸不扎手）、无气泡、无脱落
- [ ] 颜色均匀一致
- [ ] 墙地交接处、阴阳角美缝完整无断点
- [ ] 美缝剂无污染砖面（如有应及时用专用清洁剂清理）

## 四、常见验收问题

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 地砖中心区域空鼓 | 🔴 严重 | 撬起空鼓砖→清理基层→重新铺贴（注意同色号备砖） |
| 闭水试验楼下有渗漏 | 🔴 严重 | 找到漏水源→铲除该区域防水→重新涂刷防水→重新闭水试验48h |
| 卫生间地面坡度错误（水不流地漏或反向流） | 🔴 严重 | 撬起地砖→重新找坡→重新铺贴 |
| 大面积空鼓（>20%面积） | 🔴 严重 | 大面积返工：拆除→清理基层→重铺 |
| 墙砖局部空鼓（<5%面积，且非中心） | 🟡 中等 | 注浆修复（用注浆器注入瓷砖胶或环氧树脂填补空鼓层） |
| 地砖平整度偏差>3mm，两砖高低差明显 | 🟡 中等 | 撬起低洼或高凸砖重新铺贴找平 |
| 阳角崩瓷（海棠角边缘崩缺） | 🟡 中等 | 用瓷砖修补膏修补崩瓷处（或更换崩瓷砖） |
| 少量美缝漏填或气泡 | 🟢 轻微 | 补填美缝 |
| 十字卡或调平器未清理干净 | 🟢 轻微 | 清理干净 |
