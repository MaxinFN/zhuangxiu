## 竣工验收——最终全面验收

竣工验收是装修的最终质量检验，依据国家标准逐项进行。验收完成后双方签字，结算尾款。

**核心依据**：GB 50210-2018（装饰装修）、GB 50209-2010（地面）、GB 50303-2015（电气）、GB 50242-2002（给排水）、GB/T 18883-2022（室内空气质量）。

---

## 一、综合验收总清单

### 1.1 墙面（全屋）

- [ ] 乳胶漆平整：强光手电贴墙侧照无波浪纹、无砂眼
- [ ] 无裂缝：各空间墙面和顶面
- [ ] 壁纸/壁布：无翘边、无起泡、无空鼓
- [ ] 阴阳角顺直：目视+直角尺
- [ ] 踢脚线与墙面贴合无缝隙（手摸一圈）
- [ ] 开关插座周边漆面完整

### 1.2 地面（全屋）

- [ ] 地砖空鼓率：空鼓锤逐块敲，中心区域无空鼓、整面≤5%
- [ ] 地砖平整度：2m靠尺≤2mm
- [ ] 地板全屋走动无咯吱异响、无起伏
- [ ] 地板表面无划痕、无磕碰
- [ ] 卫生间/阳台地面坡度向地漏≥1%（泼水+乒乓球测试）

### 1.3 门窗（全屋）

- [ ] 每扇门自停测试（任意角度松手门扇不自行飘动）
- [ ] 门扇缝隙均匀（侧+上2-3mm、底5-8mm）
- [ ] 门锁开关顺畅（每扇≥5次）
- [ ] 密封条A4纸夹测法通过
- [ ] 窗户开关顺畅无卡阻、关闭后无漏风感

### 1.4 水电（全屋）

- [ ] 每个插座相位检测仪测试接线正确、有电
- [ ] 每个RCBO按下测试按钮能跳闸
- [ ] 每盏灯具正常点亮、双控/多控正常
- [ ] 每个水龙头出水正常、冷热方向正确
- [ ] 所有角阀/软管接口干纸巾测试无渗漏
- [ ] 热水器点火正常、出热水稳定
- [ ] 每个排水口（地漏/面盆/水槽/马桶/浴缸/洗衣机）排水通畅
- [ ] 马桶冲水有力、底座无渗漏

## 二、分空间逐项验收

### 2.1 厨房

- [ ] 橱柜：每扇门开关3次顺畅、门缝均匀；所有抽屉全拉出顺畅
- [ ] 台面：接缝平顺、水平≤2mm/m
- [ ] 水槽：满水排净、各接口无渗漏
- [ ] 烟机：各档正常、烟管接出室外/公共烟道、止逆阀有效
- [ ] 燃气灶：点火正常、火焰蓝色、熄火保护有效
- [ ] 瓷砖空鼓率合格、美缝完整

### 2.2 卫生间

- [ ] 排水坡度：水流全部流向地漏无积水
- [ ] 马桶：冲水有力、底座无渗漏
- [ ] 花洒：各模式正常、接口无渗漏、冷热方向正确
- [ ] 浴室柜：牢固、排水畅、存水弯有水封
- [ ] 浴霸：各功能正常、排气管接出室外
- [ ] 淋浴房：淋浴测试5分钟门外地面干燥

### 2.3 客厅/餐厅/卧室/阳台

- [ ] 墙面/地面/门窗按综合清单逐项过检
- [ ] 阳台地漏排水通畅、防水层完好

## 三、室内空气质量检测

- [ ] 委托**CMA认证实验室**进行专业检测
- [ ] 检测标准：**GB/T 18883-2022**（封闭≥12h后检测）
- [ ] 100㎡户型建议3-5个点位（主卧、次卧、客厅）
- [ ] 检测前：关闭门窗≥12h、温度25±2°C、所有家具已到位、无活性炭/净化器等吸附物

**合格判断（GB/T 18883-2022）**：

| 污染物 | 限值 | 状态 |
|--------|------|------|
| 甲醛 | ≤0.08mg/m³ | ☐ |
| TVOC | ≤0.60mg/m³ | ☐ |
| 苯 | ≤0.03mg/m³ | ☐ |
| 甲苯 | ≤0.20mg/m³ | ☐ |
| 二甲苯 | ≤0.20mg/m³ | ☐ |

> 任一指标超标→**首要措施**：加强持续通风（开窗+工业风扇强制排风）。通风是最有效且零成本的降低浓度方法。持续通风一月后复测仍超标再考虑专业治理。

## 四、问题记录——标签法

1. 发现一处问题→贴便签+写编号（如"P-01"）
2. 手机拍照（编号入镜+远拍一张显示位置）
3. 汇总为《竣工验收整改清单》→双方签字
4. 施工方限期整改→逐项复验→确认后撕便签→拍照留档
5. 所有问题整改完毕→**签《竣工验收合格单》**→结算尾款
