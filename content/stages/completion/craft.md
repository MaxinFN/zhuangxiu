## 竣工验收的完整流程

竣工验收是装修的"质量总检"，依据国家标准逐项检查施工质量。这个阶段的认真程度决定了入住后的生活质量——**现在找出问题，比入住后再修容易一百倍**。

---

## 一、验收总流程

```
准备验收工具+验收清单 → 逐空间逐项检查 → 发现问题贴标签+拍照编号 →
汇总整改清单（双方签字） → 施工方限期整改 → 
复验逐一销号 → 全部合格双方签字确认 → 结算尾款
```

**依据标准**：
- GB 50210-2018《建筑装饰装修工程质量验收标准》
- GB 50209-2010《建筑地面工程施工质量验收规范》
- GB 50303-2015《建筑电气工程施工质量验收规范》
- GB 50242-2002《建筑给水排水及采暖工程施工质量验收规范》

## 二、分空间逐项验收

### 2.1 客厅+餐厅

| 检查项 | 工具/方法 | 合格标准 | 依据 |
|--------|---------|---------|------|
| 墙面平整度 | 强光手电贴墙侧照 + 2m靠尺+塞尺 | ≤2mm/2m，无明显波浪纹 | GB 50210-2018 |
| 墙面有无裂缝 | 目视 | 无肉眼可见裂缝 | — |
| 地砖空鼓率 | 空鼓锤逐块敲击（听声） | 单块砖≤15%，整面≤5%，中心区域不得有空鼓 | GB 50209-2010 |
| 地砖缝格平直 | 拉5m线+钢尺 | ≤2mm/5m | GB 50209-2010 |
| 踢脚线 | 手摸+目视 | 与墙面贴合无缝隙、收口平整 | — |
| 插座接线正确 | 相位检测仪逐个测试 | L/N/PE接线正确，无"缺地线""零火反" | GB 50303-2015 |
| 插座电压 | 万用表 | 220V±10%（198-242V） | — |
| 窗户启闭 | 手动开关3次以上 | 开关顺畅、无卡阻、密封条压合均匀 | — |
| 窗户密封 | 目视缝隙+A4纸夹测 | A4纸夹在窗扇与窗框之间拉动有明显阻力 | — |

### 2.2 厨房

| 检查项 | 方法 | 合格标准 |
|--------|------|---------|
| 橱柜水平度 | 激光水平仪 | ≤2mm/延米 |
| 每扇柜门开合 | 手动开关3次 | 顺畅、无异响、缝隙均匀（偏差≤1mm） |
| 抽屉抽拉 | 全拉出+全推入 | 全拉出顺畅（带阻尼的最后一段缓冲自然）、不左右晃动 |
| 台面接缝 | 手指划+手电侧照 | 平整无缝感、无凹凸感、颜色与台面一致 |
| 台面挡水 | 目视 | 台面后挡水条与墙面贴合无缝隙 |
| 水槽排水 | 槽内满水后拔塞 | 排水快（无咕噜冒泡声）、各接口用纸巾擦拭无水珠 |
| 水槽溢水口 | 注水至溢水口 | 溢水口通畅、水从溢水口排出不渗漏 |
| 烟机运行 | 开机从低速到高速 | 各档运转正常、无异常振动/噪音、照明灯亮 |
| 瓷砖空鼓 | 空鼓锤逐块敲 | 同客厅标准 |
| 吊顶 | 推压铝扣板 | 不松动、无下垂、检修口可正常开启 |

### 2.3 卫生间

| 检查项 | 方法 | 合格标准 |
|--------|------|---------|
| 地砖坡度 | 乒乓球测试+泼水测试 | 乒乓球滚向地漏方向，泼水后水流全部流向地漏（无局部积水） |
| 地漏排水 | 倒10L水快速注入 | 排水通畅，5秒内排完水面、无冒泡倒灌 |
| 马桶冲水 | 反复冲水5-8次 | 每次冲净、冲水后水箱正常补水、无长流水 |
| 马桶周围密封 | 冲水后用干纸巾擦拭底座四周 | 纸巾干燥无湿痕 |
| 花洒出水 | 各模式切换 | 出水平稳、切换顺畅、关水后无滴漏 |
| 花洒软管接口 | 干纸巾包裹 | 纸巾干燥 |
| 浴室柜排水管 | 手摸+目视 | 存水弯接口无渗漏（存水弯水封高度≥50mm） |
| 镜柜/镜子 | 目视+手摸 | 安装牢固、镜面清晰无划痕、灯带正常 |
| 浴霸 | 各功能分别测试 | 取暖/换气/吹风/照明各功能正常 |
| 排气扇/排气道 | 开机后在排气口用纸巾测试 | 纸巾被吸住=排气通畅（非排入吊顶内） |
| 排气管道是否通向室外 | 目视 | 硬质管道→公共排风道或室外（禁止排入吊顶内） |
| 淋浴房密封 | 淋浴测试5分钟 | 门外地面干燥（玻璃门下滑轨处无渗水） |

### 2.4 卧室

| 检查项 | 方法 | 合格标准 |
|--------|------|---------|
| 地板踩踏 | 全屋缓慢走动 | 无咯吱声响、无起伏晃动 |
| 地板接缝 | 目视 | 接缝紧密、无起拱 |
| 踢脚线与墙面 | 手摸+目视 | 贴合无缝隙 |
| 门扇自停 | 门开到任意角度松手 | 门扇静止不自行飘动（说明门框垂直度合格） |
| 门扇缝隙 | 塞尺 | 侧+上各2-3mm均匀，底部5-8mm |
| 门锁 | 开关5次以上 | 锁舌对准锁扣板、开关无卡顿 |
| 窗户启闭 | 同客厅 | 同客厅标准 |
| 墙面乳胶漆 | 强光手电侧照 | 无波浪纹、无裂缝、无起皮、无砂眼 |
| 阴阳角垂直 | 直角尺 | ≤3mm |

### 2.5 阳台

| 检查项 | 方法 | 合格标准 |
|--------|------|---------|
| 地漏排水 | 倒水测试 | 排水通畅 |
| 防水（生活阳台） | 目视 | 下水管周防水层完整 |
| 栏杆/护栏 | 手摇晃 | 牢固无松动 |

## 三、水电专项验收

### 3.1 强电

- [ ] 每个插座用相位检测仪测试：接线正确、电压正常
- [ ] 配电箱内每个RCBO（漏电保护器）按下"测试（T）按钮"后立即跳闸（测试漏保功能）
- [ ] 每月应测试一次漏保（告知业主）
- [ ] 总开关（2P隔离开关）断开后全屋电路无电（用相位检测仪验证）
- [ ] 每个灯开关控制对应的灯具，无错位
- [ ] 双控开关（卧室进门+床头）双向控制正常
- [ ] 所有插座和开关面板水平、紧贴墙面、无歪斜

### 3.2 弱电

- [ ] 每个网线端口用**网络测线仪**测试：8芯全通、线序正确（T568B标准）
- [ ] 弱电箱内光猫/路由器位置有电源、有散热空间
- [ ] 电视同轴线接口连通（如有）

### 3.3 给排水

- [ ] 所有水龙头出水正常（冷热方向正确：左热右冷）
- [ ] 所有水龙头和角阀接口用干纸巾包裹检查：无渗水
- [ ] 热水器点火测试：热水出水正常、温度稳定
- [ ] 所有排水口（洗脸盆、水槽、地漏、马桶）排水通畅
- [ ] 马桶冲水有力、无回溅、水箱补水正常

## 四、问题记录与管理——"标签法"

这是最高效的验收问题追踪方法：

1. **发现一处问题**：在问题位置贴便签纸，标注序号（如"A-01"）
2. **拍照**：便签编号入镜，同时拍一张远一点的照片显示位置
3. **记录**：在验收清单表上对应编号描述问题（如"A-01：客厅沙发背景墙右侧距地1200mm处有砂眼，直径约3mm"）
4. **汇总**：将所有问题汇总为《竣工验收整改清单》，业主和施工方双方签字
5. **整改**：施工方对照清单逐项修复
6. **复验**：整改完毕后逐项复验，确认修复后撕掉对应便签、拍照留档
7. **结清**：所有问题确认整改完毕→双方签《竣工验收合格单》→结算尾款

## 五、文档归档清单

竣工验收完成后，以下文档应永久保存（建议纸质+电子扫描双备份）：

| 序号 | 文档 | 用途 |
|------|------|------|
| 1 | 管线走向照片/视频（封槽前拍摄） | **最重要！** 日后任何墙面打孔前参照 |
| 2 | 竣工图纸（含所有变更） | 了解实际管线走向和墙体结构 |
| 3 | 各阶段验收记录（双方签字） | 具有合同法律效力 |
| 4 | 产品保修卡/质保书 | 按品类分文件夹（电子版+纸质版） |
| 5 | 购买凭证/发票 | 维修和维权凭据 |
| 6 | 竣工验收合格单 | 双方签字确认，最终依据 |
| 7 | 物业装修验收合格单 | 退还装修押金的依据 |

> 管线照片的拍摄务必在瓷砖铺贴前完成。水电改造封槽前把手机放在管线旁（画面中能看到卷尺参照尺寸），对每一面墙、每一处地面逐段拍摄——这些照片的价值在未来的岁月里无可替代。
