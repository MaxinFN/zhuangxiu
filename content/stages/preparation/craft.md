## 前期准备阶段的工艺知识

虽然还没正式开工，但提前了解关键工艺概念和标准，能让你和设计师、施工方有效沟通，避免被忽悠。这一阶段的核心是**精确测量**和**方案规划**。

---

## 一、量房工艺

精确量房是后续所有设计和预算的基础，尺寸误差超过20mm就可能导致定制柜体安装不上、瓷砖排版偏差。

### 1.1 量房流程

1. **工具准备**：激光测距仪（推荐徕卡D2/博世GLM50）+ 5m卷尺 + A3方格纸或iPad测绘图软件
2. **测量顺序**：从入户门开始，顺时针逐个空间测量，避免遗漏
3. **测量内容**：
   - 每个空间的长、宽、净高（取3点平均值，标注最低点）
   - 门洞：宽×高×墙厚（标明开启方向）
   - 窗洞：宽×高×离地高×窗台深度
   - 梁：宽×高×距地高度（梁底标高）
   - 柱/管井：平面位置和突出尺寸
   - 强弱电箱：位置、尺寸、距地高度
   - 给排水立管/地漏/下水口：位置、管径
   - 烟道/排气道：平面位置和尺寸
   - 燃气表：位置和距地高度
4. **结构标注**：在图纸上用不同颜色区分 **承重墙**（黑色填充/不能动）、**非承重墙**（空心/可拆除）、**剪力墙**（通常厚度≥200mm，不能动）

### 1.2 户型图识读

| 图例 | 含义 | 对装修的影响 |
|------|------|------------|
| 黑色填充墙体 | 承重墙/剪力墙 | 拆除、开槽均禁止 |
| 空心/无填充墙体 | 非承重隔墙 | 可拆除、可开门洞 |
| 虚线 | 梁（上方）或隐藏管线 | 梁下不能打孔，吊顶需避让 |
| "X"标记 | 烟道/管井 | 位置不可移动 |
| 下沉区域 | 卫生间沉箱 | 回填方式和材料影响防水和排污 |

## 二、装修风格与施工工艺的关系

不同风格对施工精度和收口工艺有不同要求，设计阶段就应了解并考虑预算：

| 风格 | 精度要求 | 关键工艺节点 | 对材料的影响 |
|------|---------|------------|------------|
| 现代简约 | 极高（灯光侧照无波浪纹） | 墙面平整度≤2mm/2m、阴阳角完全顺直、无踢脚线做法 | 高品质乳胶漆（优等品）、极窄/隐藏踢脚线 |
| 日式/原木风 | 高（自然质感要求高） | 木作收口精细（45°拼角、无钉眼）、木蜡油涂刷 | 实木/实木贴皮、障子纸、榻榻米 |
| 法式/欧式 | 中高（线条和灯盘安装） | 石膏线安装、灯盘定位、护墙板拼装 | PU/石膏线条、实木护墙板、壁纸 |
| 工业风/侘寂 | 特殊（刻意保留肌理） | 裸墙处理、微水泥施工、金属件做旧 | 清水混凝土保护剂、微水泥、铁艺 |
| 新中式 | 高（对称和留白讲究） | 木格栅安装、大理石纹理对版、铜件收口 | 实木格栅、天然大理石、黄铜五金 |

## 三、设计方案中的工艺考量

### 3.1 瓷砖排版规划

在设计阶段就应要求设计师出具 **排砖图**（瓷砖排版图），避免施工时随意裁切导致窄条砖、不对称等问题。

- 起铺点：一般从入户门或客厅中心线开始
- 避免窄条：裁切后的砖宽度≥原砖宽度的1/3（如800mm砖裁后≥267mm）
- 墙地对缝：墙砖和地砖的缝对齐（视觉效果高级，但会增加损耗率至8-10%）
- 铺贴方向：顺光方向铺贴（与主采光窗方向平行），视觉更流畅

### 3.2 吊顶设计

| 吊顶类型 | 下吊高度 | 适用层高 | 对灯具的影响 |
|---------|---------|---------|------------|
| 全屋平顶 | 80-120mm | ≥2.7m | 可嵌任意灯具 |
| 四周边吊 | 200-300mm | ≥2.6m | 灯带+筒灯组合 |
| 不吊顶（原顶刷漆） | 0mm | 不限（尤其适合<2.6m） | 只能用明装灯具或轨道灯 |
| 悬浮吊顶 | 150-200mm | ≥2.7m | 灯带实现悬浮光影效果 |

> 层高不足2.6m的户型，不建议做全屋吊顶。灯具走明线或开浅槽（注意：承重墙顶面不能开槽）。

### 3.3 水电点位规划

设计阶段必须根据家具布局精确规划水电点位（位置+高度+数量）：

- 厨房台面插座间距≤1200mm（小家电长度一般在300-400mm，保证2-3个小家电共用一个双位插座）
- 床头插座中心距地700-800mm（床头柜高450-550mm + 预留空间）
- 电视墙预埋Φ50mm PVC管用于穿线（电视电源线、HDMI线从管内走→隐藏式安装）
- 卫生间镜前灯线、智能马桶电源、电热毛巾架电源应提前预留

## 四、与施工方沟通的工艺术语

了解以下术语有助于与工长/设计师有效沟通，避免理解偏差：

| 术语 | 解释 |
|------|------|
| 找平 | 用水泥砂浆或石膏将不平的墙面/地面处理平整 |
| 归方 | 使墙角成90°直角（阴阳角找直） |
| 冲筋 | 用石膏或水泥在墙面上做竖向筋条，作为找平的基准 |
| 海棠角 | 瓷砖阳角45°磨边拼接，留2mm缝（不割手且美观） |
| 薄贴法 | 用瓷砖胶（3-6mm厚）取代水泥砂浆（15-30mm厚）的贴砖工艺 |
| 墙压地/地压墙 | 墙面砖压在地面砖上（墙压地）更美观、防水更好 |
| 活线工艺 | 电线穿管后可以抽动更换（管内预留空间、转弯用弯头） |
| V型槽 | 石膏板接缝处切V型槽，嵌入嵌缝膏防开裂 |
| 七分安装三分材质 | 强调安装质量对最终效果的决定性作用 |

## 五、施工前准备

### 5.1 办理装修手续

- 小区物业装修许可证（含装修押金，通常¥2000-5000）
- 与物业确认：垃圾堆放点、施工时间（通常8:00-12:00/14:00-18:00，周末可能限制噪音施工）
- 与邻居沟通：提前告知装修时间段，减少投诉

### 5.2 现场保护

- 公共区域（电梯、走廊）铺设保护膜或石膏板
- 入户门、窗户用保护膜覆盖
- 下水管口用堵盖或胶带密封（防止建筑垃圾掉入堵塞）
- 燃气表用保护套包裹
