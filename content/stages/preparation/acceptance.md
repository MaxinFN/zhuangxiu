## 前期准备阶段的"验收"

这一阶段没有施工验收，但有**决策验收**——确保每一步准备到位后再开工。前期准备的质量决定了整个装修的顺利程度。

---

## 一、量房与设计验收

- [ ] 户型图精确绘制（含所有空间的长×宽×净高，多处测量的最小值）
- [ ] 承重墙/非承重墙/剪力墙已在图纸上明确标注（用不同颜色区分）
- [ ] 梁、柱、管井的平面位置和尺寸已标注
- [ ] 强弱电箱、燃气表、上下水立管位置已标注
- [ ] 所有门窗洞口的宽×高×距地高已标注
- [ ] 各空间面积已计算（用于后续材料用量估算）

## 二、装修风格与需求确认

- [ ] 装修风格已确定（家庭成员达成一致，不是某一个人的偏好）
- [ ] 已收集参考图≥20张（每个空间至少3-5张）
- [ ] 每个空间的功能需求已明确列出（如：书房需容纳2人同时办公、儿童房需预留书桌和活动区）
- [ ] 特殊需求已列表（如：收纳需求>60双鞋、需要展示柜、宠物活动区、无障碍设计）
- [ ] 与设计师沟通时已出示参考图和需求清单

## 三、预算确认

- [ ] 总预算已确定（基础施工+主材+软装+电器，含10-15%应急金）
- [ ] 各阶段预算分配已细化到具体分项
- [ ] 装修档次已确定：
  - 经济型：¥800-1200/㎡（基础装修，适合出租或预算紧张）
  - 舒适型：¥1200-2500/㎡（主流家装，材料中档+部分品牌）
  - 品质型：¥2500-4000/㎡（品牌材料+个性化设计+品质施工）
  - 高端定制：¥4000+/㎡（进口品牌+定制化+复杂工艺）
- [ ] 确认了自己的"不能省"清单（如水电网材料、防水、五金）和"可以省"清单（如软装后期补充）

## 四、施工方选择验收

- [ ] 至少对比了3家施工方（装修公司/工长/独立施工队）
- [ ] **实地查看了每家施工方的在施工地**（不是只看样板间——样板间是广告，工地才是真实水平）
- [ ] 现场观察：水电走线是否规范、瓷砖空鼓情况、墙面平整度、工地整洁度
- [ ] 与工地的业主聊过施工体验（获取真实评价）
- [ ] 核实了施工方的营业执照和资质

## 五、合同验收（开工前最后一道关）

依据 GF-2000-0207《家庭居室装饰装修工程施工合同》示范文本：

- [ ] 合同附件《材料供应清单》中每项材料的 **品牌、系列、型号、规格、颜色、数量、单价** 全部填写完整（不可有"同等档次""同类型替代"等模糊表述）
- [ ] 《施工图纸》齐全：平面布置图、拆改图、水电定位图、吊顶图、立面图、关键节点大样图
- [ ] 付款方式为分期（推荐"3-3-3-1"：签合同30%→水电验收30%→木工/瓦工验收30%→竣工验收10%），杜绝一次性付全款
- [ ] 合同中明确了增项管理：超过合同金额X%（通常5%）的增项须经双方书面确认
- [ ] 保修条款明确：水电隐蔽工程保修≥5年，饰面工程保修≥2年（这是法定最低标准）
- [ ] 工期明确：开工日期、竣工日期、延期违约金（通常按日计算）
- [ ] 合同中无"按实结算"等开口条款（或明确约定上限不超过合同金额的X%）

## 六、手续与现场准备

- [ ] 小区物业装修许可证已办理（装修押金通常¥2000-5000，验收合格后退还）
- [ ] 与物业确认：装修垃圾堆放点、施工时间段（通常8:00-12:00/14:00-18:00，周末限制噪音施工）
- [ ] 已通过书面或当面告知邻居装修时段和预计工期
- [ ] 临时水电已接通：施工用电箱（含漏保）、施工用水点
- [ ] 公共区域保护：从电梯口到入户门的走廊和墙面已铺设保护膜
- [ ] 入户门、窗户已用保护膜覆盖
- [ ] 下水管口已用堵盖密封（防建筑垃圾掉入堵塞）
- [ ] 燃气表已用保护套包裹

## 🚨 开工前的"红灯"信号

出现以下任一情况，**不要开工**：

| 红灯信号 | 为何危险 |
|---------|---------|
| 总预算还没确定 | 边做边加钱，很容易超支失控 |
| 合同中有模糊条款（"同档次替换""按实结算"等） | 材料被偷换、价格被透支 |
| 施工方报价远低于市场价（低于平均值20%以上） | 大概率后期大量增项或偷工减料 |
| 要求一次性付全款 | 失去了对施工方的全部制衡手段 |
| 没有实地看过该施工方的在施工地 | 样板间=广告；工地=真相 |
| 施工方不愿提供过往客户联系方式 | 大概率是怕被差评 |
| 材料清单没有全部填完就签了合同 | 后续发生纠纷时无从举证 |
