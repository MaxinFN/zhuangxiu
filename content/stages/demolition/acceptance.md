## 拆改阶段验收清单

### 🧱 墙体拆除验收

- [ ] 拆除范围与设计图纸一致，无多拆、漏拆
- [ ] 承重墙、梁、柱、剪力墙完好无损
- [ ] 拆除边界切割整齐，无锯齿状边缘
- [ ] 拆除后的墙体断面无松动砖块
- [ ] 建筑垃圾已装袋，未堆积在楼板上（楼板承重有限！）

### 🧹 铲墙皮验收

- [ ] 墙面铲除到位：铲到水泥层（或合同约定深度）
- [ ] 无残留的疏松腻子层
- [ ] 墙面无起皮、空鼓
- [ ] 阴阳角（墙角）的旧墙皮也清理干净

### 🏗️ 新建墙体验收（核心）

#### 砌筑质量
- [ ] 墙体垂直：用2米靠尺检查，偏差≤3mm
- [ ] 墙面平整：用2米靠尺检查，偏差≤3mm
- [ ] 灰缝饱满均匀，无透光缝隙
- [ ] 顶部斜砌工艺到位
- [ ] 新旧墙交接处已植筋（可要求查看植筋孔或拍照）

#### 挂网抹灰
- [ ] 新旧墙交接处已挂钢丝网，宽度≥200mm
- [ ] 抹灰层无空鼓：用空鼓锤敲击，声音清脆为正常，声音空洞为空鼓
- [ ] 抹灰表面平整光洁，无起砂
- [ ] 养护时间足够（至少3天以上，最好7天）

### 📐 门洞/窗洞验收

- [ ] 门洞宽度、高度符合设计尺寸（注意预留门套厚度）
- [ ] 门洞两侧墙体垂直、平行
- [ ] 门洞上方有过梁（预制混凝土或角钢），不是直接用砖砌

### 🚛 垃圾清运验收

- [ ] 建筑垃圾已全部清运出小区
- [ ] 公共区域（楼道、电梯）无遗留垃圾和灰尘
- [ ] 如有损坏公共设施，已与物业沟通修复方案

### 🚨 常见验收问题

| 问题 | 严重程度 | 如何处理 |
|------|---------|---------|
| 新墙明显歪斜（>5mm） | 🔴 严重 | 必须拆除重砌 |
| 新旧墙交接处未植筋 | 🔴 严重 | 补植筋或加钢丝网 |
| 抹灰层大面积空鼓 | 🟡 中等 | 切除空鼓区域重新抹灰 |
| 门洞尺寸偏差（<10mm） | 🟡 中等 | 修整门洞或调整门套 |
| 少量灰缝不饱满 | 🟢 轻微 | 补填灰缝即可 |
