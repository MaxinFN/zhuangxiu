## 拆改阶段验收清单

拆改验收的核心是三件事：**结构安全确认（不可拆的没动）、拆除到位（该拆的都拆了且边界整齐）、新建达标（砌筑规范+养护充分）**。

---

## 一、墙体拆除验收

**依据**：GB 50204-2015《混凝土结构工程施工质量验收规范》

- [ ] 拆除范围与设计图纸完全一致，无多拆、无漏拆
- [ ] **承重墙、剪力墙、梁、柱、阳台配重墙——完整无损，无任何敲击或开槽痕迹**
- [ ] 拆除边界切割整齐（切割线深度≥20mm的连续切线），无锯齿状撕裂边缘
- [ ] 拆除断面无松动砖块/混凝土块（用手轻推不动）
- [ ] 烟道/排气道未受任何损伤
- [ ] 拆除过程中未打断任何预埋管线（如有损坏已记录并商定修复方案）
- [ ] 建筑垃圾已全部装编织袋，**无在楼板上集中堆放**（≤500kg/处）

## 二、铲墙皮验收

- [ ] 铲除深度达标：
  - 铲至水泥砂浆层：墙面呈灰色、无明显腻子残留
  - 铲至红砖层（老房白灰墙）：墙面呈红色砖面、白灰层无残留
- [ ] 用手指抠划铲完后的墙面：无疏松层脱落
- [ ] 阴阳角的旧墙皮也已清理干净
- [ ] 厨房卫生间墙面（原瓷砖区域）铲除干净、无残留砂浆块
- [ ] 顶面如有空鼓也一并铲除（不能只铲墙面不铲顶面）

## 三、新建墙体验收（核心）

**依据：GB 50203-2011《砌体结构工程施工质量验收规范》**

### 3.1 植筋（隐蔽工程，必须拍照）

- [ ] 植筋已按竖向间距≤500mm施工（已拍照记录）
- [ ] 钢筋规格为Φ6mm或Φ8mm HRB400螺纹钢
- [ ] 入旧墙深度≥100mm
- [ ] 植筋胶饱满（孔口有胶溢出印记）
- [ ] 出墙长度≥500mm（末端有180°弯钩）

### 3.2 砌筑质量

| 检查项 | 工具/方法 | 合格标准 |
|--------|---------|---------|
| 墙体垂直度 | 2m靠尺或激光水平仪 | ≤3mm/2m |
| 墙面平整度 | 2m靠尺+塞尺 | ≤3mm/2m |
| 灰缝厚度 | 钢尺测量 | 8-12mm（水平缝和竖向缝一致） |
| 灰缝饱满度 | 目视 | 无透光缝隙（用手电筒从墙一侧照，另一侧观察无光透过） |
| 错缝搭接 | 卷尺 | ≥1/3砖长 |
| 顶部斜砌 | 目视 | 最上一皮砖斜着砌入（与顶面夹角45°-60°），无空隙 |
| 门洞过梁 | 目视 | 门洞上方有预制混凝土过梁或双角钢，两端入墙≥150mm |
| 门洞尺寸 | 卷尺 | 宽×高与设计图一致（预留门套安装厚度） |
| 门洞两侧墙体垂直 | 2m靠尺 | 平行且垂直，偏差≤3mm |
| 窗洞尺寸 | 卷尺 | 与设计一致 |

### 3.3 挂网抹灰

- [ ] 新旧墙交接面已挂热镀锌钢丝网（丝径0.5-0.8mm，孔径10×10mm）
- [ ] 钢丝网搭接宽度：每侧≥150mm（总宽≥300mm）
- [ ] U型钢钉或保温钉固定间距≤300mm
- [ ] 抹灰层分两层（底层10-12mm→面层5-8mm）
- [ ] 抹灰表面平整光洁、无起砂（手摸不落灰）
- [ ] 空鼓率：用空鼓锤逐区敲击，声音清脆（实）合格、声音空洞（空鼓）不合格。单面墙空鼓面积≤5%
- [ ] 抹灰层养护时间≥3天（最好7天），期间有洒水痕迹

---

## 四、轻钢龙骨隔墙验收

- [ ] 天地龙骨用膨胀螺栓固定牢固，间距≤400mm
- [ ] 竖龙骨间距400mm或600mm（按石膏板规格），与天地龙骨卡紧扣牢
- [ ] 竖龙骨垂直、无扭曲变形
- [ ] 隔音棉填充饱满（无遗漏、无塌陷）
- [ ] 石膏板封板自攻螺丝间距：板边≤150mm、板中≤200mm
- [ ] 螺丝头陷入板面0.5-1mm，未打穿纸面

---

## 五、垃圾清运验收

- [ ] 装修垃圾已全部清运出小区（不在公共区域堆积）
- [ ] 公共走廊、电梯厅无遗留灰尘和碎屑
- [ ] 电梯轿厢无划痕和磕碰（如有，已与物业协商解决方案）
- [ ] 楼下临时堆放点无残留垃圾

---

## 常见验收问题处理

| 问题 | 严重程度 | 处理方法 |
|------|---------|---------|
| 新墙垂直度偏差>5mm | 🔴 严重 | 拆除重砌（偏差过大会导致后续柜体无法贴墙安装） |
| 新旧墙交接处未植筋 | 🔴 严重 | 补植筋（钻孔清理→注植筋胶→插钢筋）或加挂钢丝网 |
| 抹灰层大面积空鼓（>20%面积） | 🔴 严重 | 切除空鼓区域→清理基层→重新抹灰 |
| 承重墙被敲了（即使只是开槽） | 🔴 严重 | 立即停止→请结构工程师评估→可能需要加固修复 |
| 门洞过梁遗漏 | 🔴 严重 | 拆除门洞上方砖块→安装过梁→重新砌筑 |
| 抹灰层局部小面积空鼓（<5%面积） | 🟡 中等 | 切除空鼓区→重新抹灰 |
| 门洞尺寸偏差<15mm | 🟡 中等 | 修整门洞或定制门套加宽调整 |
| 少量灰缝不饱满 | 🟢 轻微 | 补填灰缝、抹平 |
| 墙面轻微起砂 | 🟢 轻微 | 滚涂界面剂后再进行后续工序 |
