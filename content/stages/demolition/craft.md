## 拆改施工工艺

拆改是装修中结构安全风险最高的阶段。工艺的核心三原则：**安全第一（结构不可动）、减少损伤（保留墙体保护）、新建达标（规范砌筑+养护）**。

---

## 一、墙体拆除工艺

### 1.1 拆除前准备

1. **结构确认**：对照户型结构图，在承重墙、剪力墙、梁、柱等不可拆除结构上用红色标记"禁止拆除"
2. **断水断电断气**：关闭该区域所有水电阀门和燃气阀门
3. **保护措施**：不需要拆除的门窗用保护膜+胶合板封闭，地面铺设保护层
4. **邻里告知**：拆除噪音大（冲击钻、电锤100-110dB），提前和邻居沟通施工时间

### 1.2 拆除流程与工艺

| 步骤 | 工艺要点 | 依据标准 |
|------|---------|---------|
| 弹线定位 | 用墨斗沿拆除边界双面弹线，定位误差≤10mm | — |
| 切割边界 | 用混凝土切割机沿弹线切割（深度≥20mm），使拆与不拆的界面整齐 | — |
| 分层拆除 | 从上至下、分层逐块拆除，**严禁从底部掏掘**（防突然坍塌） | GB 50204-2015 |
| 洒水降尘 | 拆除全程持续喷雾洒水，PM10浓度控制在1.0mg/m³以下 | GB 16297-1996 |
| 即时清运 | 拆除废料即时装编织袋，严禁在楼板上集中堆放（每㎡楼板活荷载标准值约2.0kN/㎡） | GB 50009-2012 |

### 1.3 禁止拆除的结构

| 结构部位 | 辨别方法 | 后果 |
|---------|---------|------|
| 承重墙 | 户型图黑色填充墙体；厚度≥240mm钢筋混凝土墙 | 拆除危及整栋楼结构安全 |
| 剪力墙 | 厚度≥200mm钢筋混凝土墙（高层建筑普遍） | 影响抗震性能 |
| 梁 | 天花板下方横向突出结构 | 打断钢筋将导致上部荷载无支撑 |
| 柱 | 屋内突出的矩形/圆形结构 | 关键竖向承重构件 |
| 阳台配重墙（墙垛） | 阳台与室内之间的矮墙 | 拆除后阳台有倾覆风险 |
| 烟道/排气道 | 厨房/卫生间角落的预制管井 | 破坏后影响整栋楼排烟功能 |

> **法规依据**：《建设工程质量管理条例》第69条——擅自变动建筑主体和承重结构的，责令改正并处以罚款（个人可处5-10万元罚款）。物业发现后会要求恢复原状，费用自理。

## 二、铲墙皮工艺

### 2.1 铲除深度的判定

开发商交付的墙面通常是工程腻子+乳胶漆，长期来看质量不可靠。铲除深度由现场检验决定：

| 原有墙面状态 | 检验方法 | 铲除深度 | 后续处理 |
|------------|---------|---------|---------|
| 腻子层空鼓、粉化 | 手指抠划掉粉、敲击有空洞声 | 铲至水泥砂浆层 | 滚涂界面剂→找平石膏→腻子 |
| 白灰层（老房子） | 遇水软化、白垩质感 | **铲至红砖层** | 水泥砂浆重新抹灰→界面剂→找平石膏→腻子 |
| 防水腻子层（较结实） | 指甲抠不动、泼水不软化 | 仅铲掉松动部分 | 滚涂界面剂→找平石膏→新腻子 |

> 检验标准：铲除后用手指抠划无疏松脱落，基层坚实、无浮灰。

### 2.2 铲除后基层处理

1. 铲至水泥层后检查基层有无裂缝：结构裂缝（宽度>0.3mm）须先注浆处理后才能进行后续工序
2. 铲至红砖层后：用1:3水泥砂浆重新抹灰（厚度15-20mm），分两层施工（底层10-12mm → 养护24h → 面层5-8mm）
3. 滚涂界面剂（墙固）：铲除后墙面干燥即可滚涂，覆盖率约8-10㎡/kg，建议涂刷2遍（第1遍表干后再涂第2遍）
4. 界面剂干燥后（约2-4h，视温湿度）方可进行下一道工序

## 三、新建墙体工艺

### 3.1 蒸压加气混凝土砌块砌筑

**执行标准**：GB 50203-2011《砌体结构工程施工质量验收规范》、GB/T 11968-2020《蒸压加气混凝土砌块》

**完整流程**：

```
放线定位 → 植筋 → 砖块润湿 → 砌筑 → 顶砖斜砌 → 挂钢丝网 → 分层抹灰 → 洒水养护
```

#### 3.1.1 植筋（新旧墙连接，最关键的隐蔽工序）

| 参数 | 标准要求 | 说明 |
|------|---------|------|
| 钢筋 | HRB400 Φ6mm或Φ8mm | 螺纹钢，入旧墙深度≥100mm |
| 竖向间距 | ≤500mm（即每2-3皮砖植2根） | 第一道距地面≤500mm |
| 钻孔直径 | 比钢筋直径大4-6mm | Φ6筋钻Φ10-12mm孔 |
| 清孔 | 用压缩空气吹+毛刷清 | 孔内无粉尘植筋胶才能粘牢 |
| 植筋胶 | A+B双组份环氧型 | 注满孔体积的2/3再插入钢筋 |
| 出墙长度 | ≥500mm（或与新墙等宽-50mm） | 末端加180°弯钩 |
| 养护 | 植筋后24h内不可砌筑 | 胶体未固化前扰动会失效 |

> 验收检查：用力晃动植筋端部，无松动、植筋胶饱满溢出孔口即为合格。这一步必须拍照留存！

#### 3.1.2 砌筑要点

| 项目 | 标准 | 检查方法 |
|------|------|---------|
| 砖块含水率 | 砌筑前1-2h浇水润湿（表面吸水深度10-15mm） | 断面看吸水深度 |
| 砂浆 | M5.0或M7.5预拌砌筑砂浆，稠度50-70mm | — |
| 灰缝 | 水平缝8-12mm、竖向缝8-12mm | 塞尺检查 |
| 砂浆饱满度 | 水平缝≥80%，竖向缝≥80% | 百格网抽查 |
| 错缝搭接 | ≥1/3砖长（600mm砖搭接≥200mm） | 卷尺 |
| 拉结筋 | 每500mm高植入2根Φ6钢筋 | 已在植筋环节完成 |
| 顶部处理 | 最上一皮砖斜砌（倾斜角45°-60°），与顶面留10-20mm空隙用砂浆填实 | 目视 |
| 垂直度 | ≤5mm/层高（约2.8m） | 2m靠尺 |
| 平整度 | ≤5mm/2m | 2m靠尺+塞尺 |

#### 3.1.3 钢丝网挂设

| 参数 | 要求 |
|------|------|
| 挂网位置 | 新旧墙交接面、不同材质墙体交接面（砖墙+混凝土） |
| 搭接宽度 | 每侧≥150mm（即总宽≥300mm） |
| 网材 | 热镀锌钢丝网，丝径0.5-0.8mm，孔径10×10mm |
| 固定方式 | U型钢钉或保温钉固定，间距≤300mm |
| 抹灰覆盖 | 钢丝网必须完全嵌入抹灰层内（不得外露） |

#### 3.1.4 分层抹灰

| 层 | 厚度 | 砂浆 | 间隔 |
|----|------|------|------|
| 底层（糙灰） | 10-12mm | 1:3水泥砂浆或预拌抹灰砂浆 | 与砌筑间隔≥3天（待墙体沉降稳定） |
| 面层（细灰） | 5-8mm | 1:2.5水泥砂浆或预拌抹灰砂浆 | 底层抹灰完成后≥24h |

#### 3.1.5 养护

- 抹灰完成后洒水养护 **≥7天**，每天2-3次
- 养护期间：不得开槽、不得刮腻子、不得暴晒、强风天气应遮盖
- 养护目的：水泥水化反应需要水分，缺水会导致强度不足和干缩裂缝

## 四、轻钢龙骨隔墙工艺

**执行标准**：GB/T 11981-2008《建筑用轻钢龙骨》

适合不承重的室内隔断。相比砖墙，优势在施工速度快、厚度薄省空间、无湿作业（不产生建筑垃圾）。

| 工艺步骤 | 要点 |
|---------|------|
| 放线 | 根据设计图在地面、顶面、墙面弹线定位 |
| 固定天地龙骨 | 沿顶/沿地U型龙骨用膨胀螺栓固定于楼板，间距≤400mm |
| 安装竖龙骨 | C型竖龙骨间距400mm（9.5mm板）或300mm（12mm板），插入天地龙骨后用钳子夹紧扣牢 |
| 穿贯穿龙骨 | 38×12mm贯通龙骨水平穿入竖龙骨（每道间距≤800mm） |
| 填充隔音材料 | 岩棉或玻璃棉（密度≥24kg/m³，厚度≥50mm），填满所有空腔 |
| 封石膏板（第一面） | 自攻螺丝间距≤200mm（板中）和≤150mm（板边），螺丝头沉入板面0.5-1mm |
| 封石膏板（第二面） | 与第一面板缝错开≥300mm（错缝防止贯通裂缝） |
| 接缝处理 | V型槽切边→嵌缝膏填充→贴接缝纸带→覆盖嵌缝膏→干燥后打磨 |
| 螺丝防锈 | 所有螺丝头点涂防锈漆 |

**隔墙性能对比**：

| 指标 | 轻钢龙骨+岩棉+双面单层12mm板 | 100mm厚加气块砖墙 |
|------|---------------------------|-------------------|
| 隔声量 Rw | ~48 dB | ~52 dB |
| 厚度 | 99mm | 120mm（含抹灰） |
| 每㎡重量 | ~25 kg | ~120 kg |
| 挂重物能力 | 需预埋衬板/膨胀螺栓 | 直接打膨胀螺栓即可 |
| 工期 | 1-2天 | 7-14天（含养护） |

## 五、结构安全注意事项

1. 新建120mm厚及以上墙体且长度>5m或高度>3m时，须加设构造柱（每3m设一个）
2. 门洞上方须设置过梁（钢筋混凝土预制或角钢），两端入墙≥150mm
3. 地面拆除瓷砖+找平层后，严禁在原楼板上用冲击钻大面积打凿（会破坏楼板结构层）
4. 拆除工作涉及外墙或共用墙（与邻居共用）时，须确认不破坏防水层和隔音层
