## 水电验收

水电是装修中最重要的隐蔽工程验收，一旦封槽贴砖出现任何问题都意味着砸墙凿地。验收时业主**必须亲自到场全程参与**，不能委托他人代劳。

---

## 一、材料验收（施工前核对）

材料进场时逐项核对，与合同附件《材料供应清单》一致：

- [ ] 电线品牌、型号（BV/BVR/ZR-BV）、规格（mm²）与合同一致
- [ ] 线管为阻燃PVC材质（管壁上印有"阻燃""V0"或"B1"标识），壁厚达标（Φ16管≥1.2mm、Φ20管≥1.3mm）
- [ ] PPR水管品牌、管径、壁厚与合同一致（管身有品牌标识和S3.2系列喷码）
- [ ] 管件（弯头、三通、直接、过桥）与管材**同一品牌同一系列**
- [ ] 强弱电箱、断路器品牌型号与合同一致
- [ ] 开关插座面板品牌系列与合同一致

---

## 二、电路验收

### 2.1 布管走线验收

- [ ] 线管横平竖直，管卡固定牢固（间距≤1000mm/直线段，≤500mm/转角处）
- [ ] 墙面上无横向开槽（如有个别横槽，长度≤300mm，且避开承重墙）
- [ ] 强弱电管平行敷设间距≥300mm，交叉处有锡纸或金属过桥包裹
- [ ] 管内电线总截面积≤管内截面积的40%：
  - Φ16mm管内≤3根2.5mm²线
  - Φ20mm管内≤3根4mm²线或≤5根2.5mm²线
- [ ] 管内无电线接头（所有接头在底盒或过路盒内）
- [ ] 线管入盒处用锁扣（杯梳）连接，保护电线不被割伤
- [ ] 穿线后管口已用胶带临时封堵

### 2.2 电线分色检查

- [ ] 黄绿双色线 = 地线（PE），无挪作他用
- [ ] 蓝色线 = 零线（N）
- [ ] 红色/黄色/绿色线 = 火线（L），同一户内火线颜色统一

### 2.3 绝缘电阻测试（必做）

| 参数 | 合格标准 | 测试方法 |
|------|---------|---------|
| 各回路对地绝缘电阻 | **≥0.5MΩ** | 500V兆欧表（摇表），每条回路逐一测试 |
| 各回路之间绝缘电阻 | ≥0.5MΩ | 同上 |

> 绝缘电阻<0.5MΩ说明某处绝缘层破损或管内潮湿，必须找到原因并修复。不测绝缘电阻就封槽等于埋下日后跳闸和漏电的隐患。

### 2.4 回路规划检查

- [ ] 厨房、卫生间、普通插座回路均已配置 **RCBO（漏电保护断路器，30mA）**
- [ ] 空调、冰箱为独立回路
- [ ] 配电箱预留≥20%空位
- [ ] 配电箱内每个断路器有清晰标签（回路名称+控制区域）
- [ ] 总开关为2P隔离开关（非漏保），各分支用RCBO/MCB

## 三、水路验收

### 3.1 布管验收

- [ ] 冷热水管间距≥150mm（中心距）
- [ ] 面向墙面时：**左热右冷**（热水管在左侧，冷水管在右侧）
- [ ] 排水管有坡度≥1%（水平管每米落差≥10mm；Φ110主管≥1.2%）
- [ ] 每个排水口下方有**存水弯**（P-trap或S-trap），水封高度≥50mm
- [ ] 地漏位置在区域最低点（目视检查无积水洼地）
- [ ] 水管走顶部分已套保温棉（PE发泡管，壁厚≥20mm，接口用胶带密封）

### 3.2 打压测试（必做，全程亲自监督）

**这是整个装修中最不能省的一项测试！**

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 试验压力 | 0.8-1.0MPa（≈8-10kg/cm²） | 为工作压力的1.5倍 |
| 持续时间 | **≥30分钟** | 品牌管（伟星/日丰）质保要求≥30min |
| 允许压降 | **≤0.05MPa** | 如从0.80MPa降至0.75MPa为合格 |
| 检查范围 | 所有接口、弯头、三通、过桥、阀门 | 用干纸巾在每个接口处反复擦拭检查 |
| 检查时间 | 打压进行中和完成后各检查一轮 | — |

**操作步骤**：
1. 关闭所有水龙头和角阀
2. 用打压泵从最低点注水加压至0.8-1.0MPa
3. 用记号笔在压力表面标记初始压力值和时间
4. 保持30分钟，期间逐个检查所有接口
5. 30分钟后压力表读数下降≤0.05MPa为合格

> 压力下降>0.05MPa=某处有微渗漏→必须找到漏点（用干纸巾逐个接口擦拭）→剪掉漏点管段重新热熔→重新打压直至合格。**不找到漏点绝不封槽。**

### 3.3 排水通水测试

- [ ] 每个排水口灌入≥10L水，水流快速排走、无咕噜声、无气泡倒灌
- [ ] 马桶排污管：倒入半桶水+几张卫生纸→快速排净、不堵塞
- [ ] 所有排水管横管部分用管道内窥镜（或强光手电+镜子）检查管内有无水泥块或异物
- [ ] 地漏灌水后5秒内排净、不积水

## 四、拍照留底（封槽前最后一步）

封槽之前必须对所有管线进行系统拍照留底。这些照片在未来的5-10年内会反复用到。

- [ ] 卷尺放在管线旁作为尺寸参照物（画面中可读出管线距墙/地的距离）
- [ ] 所有墙面管线逐段拍摄（每面墙至少一张全景+关键位置特写）
- [ ] 所有地面管线逐区域拍摄
- [ ] 配电箱内部接线拍照（回路分配和接线清晰可见）
- [ ] 所有照片按房间分类存入手机相册+云盘备份

## 常见验收问题处理

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 打压30分钟压力下降>0.05MPa | 🔴 严重 | 找到漏点→剪掉重接→重新打压直至合格 |
| 绝缘电阻<0.5MΩ | 🔴 严重 | 逐段排查绝缘破损点→更换电线或修复绝缘 |
| 排水管无坡度或反坡（水往高处流） | 🔴 严重 | 拆除重铺排水管 |
| 承重墙面长横槽（>300mm） | 🔴 严重 | 恢复墙面→改为竖向开槽 |
| 强弱电间距<300mm且未屏蔽 | 🟡 中等 | 调整间距或加锡纸/金属过桥包裹 |
| 管内电线数量超标 | 🟡 中等 | 增加线管分流 |
| 漏保未安装于厨卫/插座回路 | 🔴 严重 | 更换为带漏保功能的RCBO |
| 冷热水管接反（右热左冷） | 🟡 中等 | 在混水阀处交换软管连接（或重新接管） |
| 排水管存水弯遗漏 | 🟡 中等 | 加装存水弯 |
