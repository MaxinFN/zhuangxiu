## 水电验收 — 装修中最关键的验收

### 🔌 电路验收

#### 1. 材料验收
- [ ] 电线品牌、规格与合同一致
- [ ] 线管为阻燃PVC材质，壁厚达标
- [ ] 底盒、锁扣齐全

#### 2. 布管走线验收
- [ ] 线管横平竖直，管卡固定牢固
- [ ] 线管内电线数量合理（16mm管≤3根2.5mm²线，20mm管≤5根）
- [ ] 强弱电管间距≥300mm，交叉处有锡纸屏蔽
- [ ] 墙面上无超过300mm的横槽
- [ ] 所有电线接头在底盒内，用接线端子或焊锡连接

#### 3. 通电测试
- [ ] 用摇表测试绝缘电阻：≥0.5MΩ为合格
- [ ] 每个插座左零右火（零线在左，火线在右，地线在上）
- [ ] 漏电保护器按下测试按钮能跳闸
- [ ] 电箱回路标签清晰（知道哪个开关控制哪里）

### 🚿 水路验收

#### 1. 材料验收
- [ ] 水管品牌与合同一致
- [ ] 冷热水管区分（热水管壁更厚或有红线标识）
- [ ] 管件齐全（弯头、三通、过桥、阀门）

#### 2. 布管验收
- [ ] 冷热水管间距≥150mm
- [ ] 左热右冷（面对墙面时，热水管在左边）
- [ ] 排水管有坡度（≥1%，即1米长度下坡1cm）
- [ ] 排水管有存水弯（防臭）
- [ ] 地漏位置在最低点

#### 3. **打压测试（必做！）**

这是整个装修中最不能省的一步！

**操作方法：**
1. 用打压泵将水管加压至0.8-1.0MPa（约8-10公斤）
2. 保持压力30分钟以上
3. 观察压力表是否下降

**合格标准：**
- 30分钟内压力下降＜0.05MPa
- 所有接口、阀门无渗水
- 压力下降＞0.05MPa → 必须找到漏点并重做

> ⚠️ **亲自在场！** 打压测试时必须亲自全程看着，有的工人会敷衍甚至不做。

#### 4. 排水测试
- [ ] 每个排水口倒一桶水，观察下水速度
- [ ] 无咕噜咕噜声（有声音说明排水不畅或有堵塞）
- [ ] 地漏排水顺畅，不积水

### 📸 验收必备：拍照留底！

**封槽前必须拍照！** 把所有水电走线拍下来：

- [ ] 每个墙面的管线走向（正面照）
- [ ] 每个插座、水嘴的特写
- [ ] 配电箱内部布局
- [ ] 尺子比在管线上拍照（记录准确位置）

> **为什么拍照？** 以后墙上打孔（装挂画、装柜子）时，你知道哪里有线管，不会打穿。

### 🚨 常见验收问题

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 打压30分钟压力明显下降 | 🔴 严重 | 找到漏点，剪掉重接 |
| 强弱电未分离 | 🟡 中等 | 加锡纸屏蔽或调整间距 |
| 排水管无坡度或反坡 | 🔴 严重 | 重新铺设排水管 |
| 墙面超长横槽 | 🔴 严重 | 恢复墙面，改为竖槽 |
| 管内电线数量超标 | 🟡 中等 | 加管分流 |
