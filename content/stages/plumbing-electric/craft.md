## 水电施工工艺

水电是装修中最重要的隐蔽工程，一旦封槽贴砖，任何问题都意味着砸墙凿地。以下按国家规范和行业通用做法详述每个工艺环节。

---

## 一、水电定位

### 1.1 定位流程

```
确认家具尺寸和摆放位置 → 确认电器型号和功率 → 确定插座位置/高度 → 确定开关位置 → 确定给排水点位 → 墙面标注
```

### 1.2 标准定位高度（距完成地面，单位mm）

| 点位 | 高度 | 说明 |
|------|------|------|
| 普通插座（低位） | 300 | 家具背后/侧面，不被家具遮挡 |
| 电视机插座（低位） | 300-600 | 电视柜背后，预埋Φ50mm穿线管 |
| 电视机插座（壁挂） | 1100左右 | 电视挂墙后方（需提前确认电视尺寸和挂架位置） |
| 厨房台面插座 | 1100-1200 | 距台面完成面≥200mm（防水溅） |
| 油烟机插座 | 2100-2300 | 烟机上方吊顶内或吊柜内 |
| 冰箱插座 | 300-1500 | 冰箱侧面（方便拔插） |
| 空调插座（挂机） | 1800-2100 | 空调侧面或上方 |
| 开关面板 | 1300-1400 | 成人手掌自然按压高度（距门框150-200mm） |
| 床头插座+开关 | 700-800 | 床头柜上方50-100mm（双控走道开关） |
| 书桌插座 | 950-1050 | 桌面上方50-100mm |
| 热水器插座 | 1800-2000 | 避开淋浴区，带防水盒（IP44） |
| 洗衣机插座 | 1200 | 洗衣机侧面，带防水盒 |
| 智能马桶插座 | 300-400 | 马桶侧面，带防水盒 |
| 弱电（网线/TV） | 300 | 与强电插座并排布置，间距≥300mm |

### 1.3 定位标注方法

- 用粉笔或记号笔在墙面直接画出插座/开关的轮廓（86×86mm），标注类型（五孔/开关/网线等）
- 用水准仪或激光水平仪确保同排插座在同一水平线上
- 定位完成后拍照+录像，作为管线走向的原始记录

## 二、开槽工艺

### 2.1 开槽规范

| 规范项 | 标准要求 | 依据 |
|--------|---------|------|
| 墙面开槽方向 | **竖向开槽为主**，横向开槽≤300mm（非承重墙≤500mm） | GB 50203-2011 |
| 槽深度 | 线管外径+10mm以上（保证抹灰保护层≥10mm） | — |
| 槽宽度 | 线管外径+10-20mm（单管）/多管并排宽度+管间距5mm | — |
| 禁止开槽位置 | 承重墙/剪力墙横向开槽、梁、柱、烟道、楼板开槽 | GB 50011-2010 |
| 开槽工具 | 开槽机（角磨机+金刚石锯片）+ 电锤（严禁用大锤直接砸） | — |

> 墙面横向开槽会削弱墙体结构承载力。尤其承重墙和剪力墙的水平横槽会切断竖向钢筋，严重影响抗震性能。

### 2.2 禁止行为

- ❌ 承重墙上开深横槽（即使长度<300mm也应尽量避免）
- ❌ 梁上开槽打孔（梁内钢筋密集，打断钢筋后果严重）
- ❌ 楼板开槽（会破坏楼板结构保护层和钢筋）
- ❌ 烟道/排气道上开槽（破坏共用管道，影响全楼排烟）
- ❌ 用冲击钻在空心砖墙上直接凿（必须用切割机先切边）

## 三、线管铺设与穿线

### 3.1 线管铺设

| 规范 | 标准 | 原因 |
|------|------|------|
| 走向 | 横平竖直（地面走最近路径，墙面竖向） | 日后维修易找线 |
| 转角 | 必须用弯头或弹簧弯管器成型，**禁止直接折弯** | 折弯会压扁线管，减小内径 |
| 管卡间距 | ≤1000mm（直线段）/ ≤500mm（转角处） | 固定牢靠，抹灰不位移 |
| 强弱电间距 | **平行敷设≥300mm**，交叉处用锡纸或金属过桥包裹 | 防电磁干扰（网线信号衰减） |
| 单管电线数量 | 管内电线总截面积≤管内截面积的40% | 保证散热空间+抽线余量 |
| 管内接头 | **严禁管内接头** | 接头必须在底盒或过路盒内 |
| 管入盒 | 线管必须用锁扣（杯梳）与底盒连接 | 保护电线不被盒边割伤 |
| 管线封堵 | 穿线完成后临时用胶带封住管口 | 防止抹灰/批腻子时异物掉入 |

### 3.2 电线分色（强制性要求）

| 线色 | 功能 | 不允许错用 |
|------|------|-----------|
| 黄绿双色 | 地线（PE） | **专用色，严禁用作其他线路** |
| 蓝色 | 零线（N） | 不可用作火线或地线 |
| 红色/黄色/绿色 | 火线（L，A/B/C三相） | 同一户内火线颜色应统一（建议全部用红色） |

> **执行标准**：GB 50303-2015《建筑电气工程施工质量验收规范》和 GB 50259-96。颜色混淆是触电和电器损坏的重要原因之一。

### 3.3 穿线

- 穿线前检查线管是否通畅（用细钢丝先穿一遍）
- 用穿线器或钢丝引导电线穿入
- 穿线后每个底盒内的电线预留长度≥150mm（方便接线和面板安装）
- 配电箱内电线预留长度≥箱体半周长（一般≥500mm）

## 四、PPR水管热熔焊接

### 4.1 焊接原理

PPR（无规共聚聚丙烯）管通过热熔器将管材外表层和管件内表层同时加热到熔融温度（约260°C），然后迅速对接插入，冷却后融为一体——焊接处的强度等于甚至高于管材本身。

### 4.2 焊接流程与参数

| 步骤 | 操作要点 |
|------|---------|
| 1. 剪管 | 用PPR专用剪刀垂直剪管（断面必须平齐，偏斜角度≤1°） |
| 2. 去毛刺 | 用刮刀或砂纸去除管口内外毛刺 |
| 3. 清洁 | 用干净抹布擦拭管端和管件内壁（**不能有油污、水、灰尘**） |
| 4. 标记深度 | 用记号笔在管端标记插入深度（管件承插深度-1mm） |
| 5. 加热 | 热熔器温度260±10°C，将管端和管件同时推入加热模头 |
| 6. 加热时间 | Φ20mm→5秒 / Φ25mm→7秒 / Φ32mm→8秒 |
| 7. 对接插入 | 加热完毕后迅速拔出并直线插入到底，保持3-5秒不动，自然冷却≥2分钟 |
| 8. 冷却 | 冷却期间不可转动、不可受力、不可通水 |

### 4.3 焊接质量判定

| 合格特征 | 不合格特征 | 原因 |
|---------|-----------|------|
| 接口有均匀的双"眼皮"翻边 | 无翻边或一侧无翻边 | 加热不足或插入不到位 |
| 翻边对称、颜色均匀 | 翻边发黄、焦黑 | 温度过高或加热时间过长 |
| 管件整体无变形 | 管件变扁、内径缩小 | 加热过度或旋转扭曲 |
| 接口处紧密无缝隙 | 管端未插入到底 | 插入深度不够 |

### 4.4 水管走顶还是走地？

这是水电施工中最常见的争议，各有适用场景：

| 方案 | 优点 | 缺点 | 适用建议 |
|------|------|------|---------|
| **走顶** | 漏水可及时发现（水往下滴）；维修不需要砸地 | 用管量多30-40%；需吊顶遮挡；管道有凝结水风险（需套保温棉） | **厨卫走顶** |
| **走地** | 施工简单；省管材 | 漏水极难发现（等楼下邻居找上门已造成损失）；维修需砸地砖 | 其他干区可用 |
| **走墙** | 竖向管路必经路径 | — | 开关/水嘴的竖管走墙 |

> 推荐方案：**厨房和卫生间水管全部走顶**。厨卫本身需要吊顶，水管走顶正好隐藏在吊顶内。其他区域（阳台、设备间）根据现场情况可走地。

### 4.5 水管保温

- 走顶的热水管必须套保温棉（PE发泡管，壁厚≥20mm），减少热量损失和凝结水
- 走顶的冷水管也建议套保温棉（防凝结水滴落损坏吊顶）
- 保温棉接口处用专用胶带缠绕密封

## 五、配电系统安装

### 5.1 回路规划

| 回路 | 线径 | 断路器 | 漏电保护 |
|------|------|--------|---------|
| 照明（分1-2路） | 1.5mm² | MCB C16 1P | 不需要 |
| 普通插座（分1-2路） | 2.5mm² | RCBO C20 1P+N 30mA | **必须** |
| 厨房插座 | 4mm² | RCBO C25 1P+N 30mA | **必须** |
| 卫生间插座 | 4mm² | RCBO C20 1P+N 30mA | **必须** |
| 空调挂机（每台） | 2.5mm² | MCB C20 1P | 不需要（固定电器） |
| 空调柜机/风管机 | 4mm² | MCB C25 1P | 不需要 |
| 冰箱（独立） | 2.5mm² | MCB C16 1P | 不需要 |
| 即热式热水器 | 6mm² | MCB C32 1P | 不需要 |

### 5.2 强电箱安装

- 强电箱底边距地1500-1800mm（便于操作且儿童不易触及）
- 箱体内预留20%空位（日后增加回路）
- 每个断路器标签清晰（用标签机打印回路名称贴在对应空开上方）
- 总开关用2P隔离开关（不带漏保），各分支回路用RCBO/MCB
- 进户线≥10mm²

### 5.3 等电位联结（容易被忽视的关键安全项）

卫生间必须做局部等电位联结（LEB）。将卫生间内所有金属部件（金属给水管、金属排水管、金属浴缸支架、金属暖气管等）用≥4mm²铜导线连接到等电位端子盒。

> **依据**：GB 50054-2011《低压配电设计规范》和 GB 50303-2015。等电位联结可在漏电时显著降低接触电压，是与漏电保护器配合的生命安全双保险。

## 六、打压与测试

### 6.1 水管打压测试

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 试验压力 | 工作压力的1.5倍，且≥0.8MPa | 家装常用0.8-1.0MPa |
| 试验时间 | ≥30分钟 | 伟星/日丰等品牌质保要求≥30min |
| 允许压降 | ≤0.05MPa | 如从0.8MPa降到0.75MPa为合格 |
| 试验介质 | 水（非空气） | 水不可压缩，微漏即可见压降 |
| 检查范围 | 所有接口、弯头、三通、阀门 | 用干纸巾在每个接口处擦拭检查 |

### 6.2 排水管通水测试

- 每个排水口（地漏、洗手盆、马桶排污、洗衣机排水）灌入≥10L水
- 观察排水速度（应流畅、无咕噜声、无倒灌）
- 管道内窥镜检查（推荐，检查横管内有无水泥块或异物堵塞）

### 6.3 电路测试

| 测试项 | 方法 | 合格标准 |
|--------|------|---------|
| 绝缘电阻 | 500V兆欧表 | ≥0.5MΩ（各回路对地） |
| 插座接线 | 相位检测仪 | L/N/PE接线正确 |
| 漏电保护 | 按RCBO上的测试按钮 | 立即跳闸 |
| 通路测试 | 万用表通断档 | 每条回路从配电箱到末端插座导通 |

## 七、拍照留底（最关键的一步）

封槽和贴砖之前，用手机对全屋所有管线走向拍照+录像：

1. 用卷尺放在管线旁作为尺寸参照物（画面中可读出管线距墙/地的距离）
2. 所有墙面管线（竖向和横向）逐段拍摄
3. 所有地面管线逐区域拍摄
4. 配电箱内部接线拍照（记录回路分配）
5. 所有照片按房间分类存入手机相册和云盘

> 这些照片在未来5-10年内会反复用到：挂画打孔、安装柜体、维修水电——每一张照片都可能避免一次打穿水管的灾难。
