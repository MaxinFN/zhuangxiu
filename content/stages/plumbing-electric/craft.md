## 水电施工核心工艺

### 一、水电定位

**定位顺序：**
```
确定家具尺寸 → 确定电器位置 → 确定插座位置 → 确定开关位置 → 确定水嘴位置
```

**标准高度参考（从地面算）：**

| 功能 | 高度 | 说明 |
|------|------|------|
| 普通插座 | 300mm | 低插座，家具后面或旁边 |
| 电视墙插座 | 300-600mm | 电视柜遮挡 |
| 厨房台面插座 | 1100-1200mm | 台面上方 |
| 空调插座 | 1800-2100mm | 靠近空调位置 |
| 开关面板 | 1300-1400mm | 成人手自然下垂高度 |
| 床头插座+开关 | 700-800mm | 床头柜上方 |
| 书桌插座 | 900-1000mm | 桌面上方 |
| 热水器插座 | 1800mm | 避开淋浴区 |
| 洗衣机插座 | 1200mm | 侧面，带防水盒 |

### 二、开槽工艺

**规范：**
- 墙面尽量**竖开槽**，横槽不超过300mm（影响墙体结构）
- 地面可以横开槽
- 槽深：管径+10mm以上（保证抹灰覆盖厚度）
- 槽宽：管径+10mm左右

**严禁：**
- ❌ 承重墙开深横槽（破坏结构）
- ❌ 梁、柱上开槽
- ❌ 烟道上开槽

### 三、布管穿线

**线管铺设规范：**

1. **横平竖直** — 走线要规整，方便以后找线
2. **转角用弯头** — 不能用线管直接折弯（会压扁）
3. **管卡固定** — 线管用管卡固定于墙面，间距≤1米
4. **强弱电分离** — 弱电线（网线、电视线）与强电线管间距≥300mm，交叉处用锡纸包裹
5. **管内无接头** — 电线接头必须在底盒内，管内不允许有接头

**电线分色规范：**
- 火线（L）：红色 / 黄色 / 绿色
- 零线（N）：蓝色 / 黑色
- 地线（PE）：黄绿双色

### 四、水管焊接（PPR热熔）

**流程：**
```
剪管（平直口） → 清理管口毛刺 → 热熔器加热 → 插入连接 → 冷却定型
```

**要点：**
- 热熔温度：260°C左右
- 加热时间：管径Φ20mm加热5秒，Φ25mm加热7秒
- 插入后保持3-5秒不动，自然冷却
- 焊接质量判断：接口处有均匀的"双眼皮"翻边

**水管走顶还是走地？**

| 方案 | 优点 | 缺点 |
|------|------|------|
| 走顶 | 漏水容易发现，维修不砸地 | 用管多，需要吊顶遮挡 |
| 走地 | 施工简单，省管材 | 漏水难发现，维修要砸地 |

>推荐：**厨卫走顶，其他区域走地**。如果预算允许，全部走顶。

### 五、电路回路规划

**回路分配建议（100㎡三室）：**

| 回路 | 说明 | 空开规格 |
|------|------|---------|
| 照明1 | 客餐厅+走廊灯 | 16A 1P |
| 照明2 | 卧室+卫生间灯 | 16A 1P |
| 普通插座1 | 客厅+餐厅插座 | 20A 1P+N漏保 |
| 普通插座2 | 卧室插座 | 20A 1P+N漏保 |
| 厨房插座 | 厨房所有插座 | 25A 1P+N漏保 |
| 卫生间插座 | 含浴霸 | 20A 1P+N漏保 |
| 空调1 | 客厅空调 | 20A 1P |
| 空调2 | 主卧空调 | 16A 1P |
| 冰箱 | 单独回路 | 16A 1P |

> **漏保** = 漏电保护器，人触电时自动断电，保护生命安全。
> **卫生间、厨房、普通插座**必须用漏保！
