## 墙面施工工艺

墙面施工是全屋表面积最大的表面处理工程，工艺遵循"基层→界面→找平→腻子→底漆→面漆"的层级体系。每道工序的质量都会累积影响最终效果。

---

## 一、整体流程

```
基层处理 → 滚涂界面剂(墙固) → 钉眼防锈 → 阴阳角安装护角条 → 
嵌缝石膏填缝 → 找平石膏(粗找平) → 批刮腻子第1遍 → 干燥+打磨 →
批刮腻子第2遍 → 干燥+打磨 → 批刮腻子第3遍(精收光) → 精细打磨 →
清理浮尘 → 滚涂底漆(1遍) → 点补瑕疵 → 滚涂面漆第1遍 → 干燥 →
滚涂面漆第2遍 → 养护(关窗阴干≥7天)
```

---

## 二、基层处理

### 2.1 基层检查

| 检查项目 | 合格标准 | 不合格处理 |
|---------|---------|-----------|
| 墙面空鼓 | 无空鼓（敲击声清脆而非空洞） | 切除空鼓区→重新抹灰 |
| 裂缝 | 结构裂缝宽≤0.3mm | 宽度>0.3mm的裂缝须注浆修补（环氧树脂压力注浆） |
| 墙碱 | 无白色结晶粉末（泛碱） | 用草酸溶液清洗→清水冲洗→干燥→封闭底漆 |
| 油污/脱模剂 | 无 | 用碱水或脱脂剂清洗 |
| 浮灰 | 无 | 扫除后用潮湿抹布擦拭 |

### 2.2 滚涂界面剂（墙固）

| 参数 | 要求 |
|------|------|
| 颜色 | 墙固有颜色（常见黄色），便于检查是否漏涂 |
| 施工方式 | 滚筒滚涂，覆盖率约8-10㎡/kg/遍 |
| 遍数 | 建议2遍（第一遍表干后滚第二遍） |
| 干燥时间 | 2-4h（表干），完全固化≥24h |

> 界面剂作用：渗透封闭基层浮灰、提高腻子与墙面的粘结强度（降低空鼓脱落风险）。

### 2.3 阴阳角护角条安装

| 步骤 | 工艺 |
|------|------|
| 定位 | 用激光水平仪或线垂确定垂直线 |
| 裁剪 | 护角条按墙高裁剪（减去地面到踢脚线上沿的尺寸） |
| 固定 | 用嵌缝石膏将护角条粘贴在墙角两侧，刮平挤出多余石膏 |
| 检查 | 护角条两侧用靠尺检查，确保垂直、顺直 |

### 2.4 石膏板部位专项处理

| 处理项 | 工艺 |
|--------|------|
| 螺丝防锈 | **所有自攻螺丝头逐一点涂防锈漆**（黑色磷化螺丝的镀层在拧入时可能破损），遗漏=日后锈斑透出漆面 |
| 板缝填补 | V型槽→嵌缝石膏填缝→贴接缝纸带→覆盖嵌缝膏→干燥打磨 |
| 阴角接缝 | 吊顶与墙面交接的阴角贴接缝纸带（防止此处石膏板与墙面因材质不同而开裂） |

## 三、墙面找平

### 3.1 找平方案选择

| 墙面状态 | 适用方案 | 说明 |
|---------|---------|------|
| 平整度偏差≤3mm/2m | 直接批腻子找平（薄层） | 合格基层，腻子薄批即可 |
| 平整度偏差3-10mm/2m | 找平石膏粗找→腻子精找平 | 先用找平石膏填低谷 |
| 平整度偏差>10mm/2m | 冲筋找平（用石膏或水泥砂浆做竖向筋条作为找平基准） | 大面积不平或老房新做 |

### 3.2 冲筋找平工艺

这是精益求精的做法——在墙面上按1.2-1.5m间距用找平石膏做出竖向筋条（筋厚=找平厚度），然后用2m铝合金刮尺沿筋条刮平。冲筋后的墙面平整度可达≤2mm/2m。

## 四、批刮腻子

### 4.1 3遍腻子体系

| 遍数 | 目的 | 单遍厚度 | 刮涂方向 | 间隔条件 |
|------|------|---------|---------|---------|
| 第1遍（基层） | 大面积覆盖、弥合基层不平 | ≤1.5mm | 随意 | 完全干燥（夏季≥1天、冬季≥3天） |
| 第2遍（整体） | 填补第1遍干燥收缩产生的微坑 | ≤1mm | 与第1遍垂直 | 完全干燥 |
| 第3遍（精收光） | 最终精细找平 | ≤0.5mm | 与第2遍垂直 | 完全干燥后精细打磨 |

**腻子调配**：
- 成品耐水腻子（N型）只需加清水搅拌（按产品说明书的水粉比，通常1kg粉加0.3-0.4L水）
- 电动搅拌器搅拌至无颗粒、膏状顺滑
- 单次搅拌量以2-3h内用完为限（超过可操作时间会初凝失效）
- **严禁在腻子中加胶水或熟胶粉**（成品腻子已含足量胶粉）

### 4.2 打磨

| 项目 | 要求 |
|------|------|
| 砂纸目数 | 240#（第1-2遍）/ 320#（第3遍精磨） |
| 打磨工具 | 砂纸架+砂纸（手动）或电动打磨机+吸尘器 |
| 灯光检查 | 用手电筒或工作灯紧贴墙面侧照，不平处会投下阴影 |
| 检查标准 | 2m靠尺检查 ≤2mm（高级抹灰）/ ≤3mm（普通抹灰） |
| 清尘 | 打磨后用鸡毛掸或吸尘器彻底清除墙面粉尘（粉尘不除，底漆无法渗透结合） |

## 五、涂刷施工

### 5.1 底漆（1遍）

- **作用**：封闭腻子层碱性（水泥基腻子呈碱性，不封闭会导致面漆泛碱发花）；增强面漆附着力；减少面漆用量
- 施工：滚筒滚涂（覆盖率约10-12㎡/L/遍）
- 底漆干燥后进行**点补**：用腻子补平打磨后暴露的微小砂眼/划痕，干后局部打磨→再补一点底漆

### 5.2 面漆（2遍）

| 参数 | 要求 |
|------|------|
| 施工工具 | 优质短毛滚筒（羊毛或化纤，毛高6-8mm）——大面；羊毛刷——边角 |
| 稀释 | 按说明书（通常加水5-10%），不稀释或超量稀释都会影响成膜质量 |
| 涂布率 | 约12-14㎡/L/遍（视底材吸水率和涂刷手法差异） |
| 每遍间隔 | 指触干≥2-4h（温度20°C，湿度50%），低温高湿延长 |
| 施工方向 | 第一遍竖向→第二遍横向（交叉方向保证均匀覆盖） |
| 搭接 | 每滚搭接上一滚的1/3宽度，快干前完成一个完整墙面（避免接头干湿不均） |

### 5.3 施工环境控制

| 环境因素 | 允许范围 | 影响 |
|---------|---------|------|
| 温度 | 5-35°C（最佳15-25°C） | <5°C乳胶漆无法成膜（MFFT低温成膜阈值），会粉化脱落 |
| 基层含水率 | ≤10% | 过高导致漆膜起泡、泛碱 |
| 空气相对湿度 | ≤85% | 过高延长干燥时间、漆膜可能发白 |
| 通风 | 每遍干燥期间微通风（开窗1-2cm缝），禁止强风直吹 | 表面干太快导致内部水分应力不均→龟裂 |

### 5.4 养护

- 面漆涂刷完成后**关窗阴干≥7天**
- 养护期间避免触碰墙面、避免扬尘、避免直接加热（暖风机、暖气）
- 养护期间漆膜持续固化（乳胶漆完全固化需28天，7天后达到大部分硬度），7天后可正常使用

## 六、特殊工艺

### 6.1 分色墙

1. 先涂刷浅色区域（全墙到顶）
2. 浅色漆完全干燥后，用**低粘性美纹纸**（黄色和纸胶带）沿分界线贴好
3. 深色区域滚涂→**漆半干时（约15-30min）沿45°方向撕除美纹纸**——全干后撕会把漆皮一起扯下来
4. 分色线应选择在阴角、阳角或墙面中部水平线（用激光水平仪打出精确水平线）

### 6.2 艺术漆施工

- 艺术漆施工需要专业师傅，工费约 ¥100-300/㎡
- 效果取决于师傅手艺和审美，务必先看工地实际样品
- 修补困难（后期磕碰修补后颜色和纹理很难完全一致）

### 6.3 微水泥施工

- 基层必须是混凝土/水泥砂浆/水泥纤维板等刚性基层（不能做在石膏板或木基层上）
- 施工层序：底涂层→微水泥基层（含玻纤网）→微水泥面层→封闭底漆→聚氨酯面漆（2遍）
- 每道涂层间隔≥12h
- 最终面漆需养护≥7天才能正常使用
