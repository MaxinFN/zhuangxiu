## 墙面验收

墙面占全屋表面积的60-70%，是最影响视觉效果的部分。墙面验收的核心是**平整度（灯光侧照法）**和**漆膜质量**。

**依据**：GB 50210-2018《建筑装饰装修工程质量验收标准》

---

## 一、基层验收（腻子打磨后、刷漆前）

这个阶段的验收决定了最终面漆效果——腻子不平，再好的漆也白费。

| 检查项 | 工具 | 合格标准 |
|--------|------|---------|
| 墙面平整度 | 2m靠尺+塞尺 | ≤2mm/2m（高级抹灰）/ ≤3mm/2m（普通抹灰） |
| 阴阳角方正度 | 直角尺+塞尺 | ≤3mm |
| 阴阳角顺直 | 目视或激光水平仪 | 一条直线、无明显弯曲 |
| **砂眼/坑洞** | 强光手电贴墙侧照 | 无肉眼可见的坑洞或粗砂痕迹 |
| **波浪纹** | 强光手电贴墙侧照+平行墙面移动 | 无波浪状阴影（手电光照下墙面光斑应平滑过渡） |
| 打磨痕迹 | 目视 | 无明显砂纸打磨的粗痕 |
| 浮尘 | 手摸 | 手掌摸墙面无白粉残留（浮尘不除底漆无附着力） |
| 护角条 | 目视 | 无外露、嵌入腻子内、顺直 |
| 踢脚线部位 | 2m靠尺 | 平整（踢脚线安装后与墙面贴合无缝隙） |

### 灯光检查法（最重要）

用手电筒或强光工作灯**紧贴墙面侧照**，灯光平行于墙面。平整的墙面→光斑均匀平滑过渡。墙面缺陷会投下明显阴影：
- 波浪纹=腻子打磨不均匀
- 黑点/坑洞=砂眼或粗砂粒未填
- 鼓包=基层空鼓或腻子层过厚

---

## 二、底漆验收

- [ ] 底漆已涂刷（不可省略！）——检查方式：滚筒痕迹和颜色与腻子层有差异
- [ ] 底漆无漏涂、无流挂、无气泡
- [ ] 底漆干燥后**点补**了打磨后暴露出的微小瑕疵（砂眼补腻子→局部打磨→补底漆）

---

## 三、面漆验收

### 3.1 涂刷质量

| 检查项目 | 合格标准 | 检查方法 |
|---------|---------|---------|
| 颜色 | 与色卡选定颜色一致，无色差 | 自然光下1m距离目视（不同光照下色差感受不同，以色卡标准光照为准） |
| 色泽均匀度 | 不发花、无明显深浅不一 | 正面观察（非侧光） |
| 刷纹/滚筒痕 | 无明显滚筒痕迹和刷子印痕 | 正面和斜视 |
| **流挂** | 无泪滴状漆痕 | 目视 |
| **起皮/脱落** | 无 | 目视+手指轻抠（禁止用力抠扯） |
| **气泡/针孔** | 无 | 目视 |
| 裂纹 | 无 | 目视（重点检查石膏板接缝处、新旧墙交接处） |
| 光泽度 | 均匀一致（哑光/蛋壳光/半光与选定一致） | 目视 |
| 颗粒/异物 | 无嵌入漆膜的粉尘颗粒 | 手摸+目视 |
| 分色线（如有分色） | 界线清晰、整齐、无相互渗透 | 目视（重点检查分色线两个颜色交界处） |

### 3.2 覆盖与完整

- [ ] 顶面、所有墙面全覆盖无遗漏（尤其检查窗帘盒内侧、吊顶灯槽内侧等隐蔽位置）
- [ ] 开关插座面板周边的漆面完整（面板拆除后补漆）
- [ ] 门套、窗套与墙面交接处的漆面完整

---

## 四、特殊墙面材料验收

### 4.1 壁纸/壁布

- [ ] 无翘边、无起泡、无空鼓
- [ ] 花纹图案对版（两幅拼接处图案完整对接）
- [ ] 拼缝≤0.5mm（站在1m外看不见拼缝）
- [ ] 无胶水溢出污染表面
- [ ] 壁布无褶皱

### 4.2 艺术漆

- [ ] 纹理效果与样品一致
- [ ] 纹理均匀、无断纹
- [ ] 颜色与选定一致

### 4.3 硅藻泥

- [ ] 纹理均匀、厚度一致
- [ ] 无脱粉（手摸表面不粘白粉）
- [ ] 颜色均匀

---

## 五、养护条件检查

刷漆完成后至验收前：

- [ ] 面漆涂刷完成后已关窗养护≥7天（微通风开1-2cm缝即可，禁止强风直吹）
- [ ] 养护期间室内无扬尘（如打磨/切割等施工作业）
- [ ] 养护期间未触碰墙面

## 常见验收问题

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 灯光侧照有明显波浪纹 | 🔴 严重 | 重新精细打磨→补腻子→重新刷底漆+面漆 |
| 面漆起皮/脱落 | 🔴 严重 | 铲除起皮区域→检查基层（可能是有潮气或基层疏松）→重新处理→重新刷漆 |
| 大面积色差/发花 | 🟡 中等 | 加刷一遍面漆（确保同一批次漆） |
| 分色线渗色、不清晰 | 🟡 中等 | 用美纹纸重新贴线→补刷边缘 |
| 砂眼/坑洞（面积少、数量不多） | 🟡 中等 | 点补腻子→打磨→补底漆→补面漆 |
| 少量流挂 | 🟢 轻微 | 干燥后轻轻打磨→补面漆 |
| 开关插座周边漆面不完整 | 🟢 轻微 | 摘下面板→补刷周边 |
| 漆面有嵌入灰尘颗粒 | 🟢 轻微 | 干燥后轻轻打磨→补面漆 |
