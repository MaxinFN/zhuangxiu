## 墙面验收 — 一摸二看三照

### 🔦 灯光验收法（最重要）

**工具：** 强光手电筒或手机手电筒

**方法：**
1. 关闭所有照明
2. 手电筒贴着墙面（距离墙面10cm以内）
3. 从不同角度照射墙面

**合格标准：**
- 无明显波浪纹（灯光下墙面平整顺滑）
- 无明显的打磨痕迹（一圈一圈的砂纸印）
- 无针孔、气泡

### ✅ 验收清单

#### 腻子层验收（刷漆前）

- [ ] 墙面平整：2米靠尺检查，偏差≤2mm
- [ ] 阴阳角顺直：偏差≤2mm
- [ ] 打磨均匀，无砂纸漏打
- [ ] 无起泡、无脱落、无空鼓
- [ ] 浮灰已清扫干净

#### 底漆验收

- [ ] 底漆涂刷均匀，无漏刷
- [ ] 底漆干透后再刷面漆（通常间隔2-4小时）

#### 面漆验收（最终）

**看：**
- [ ] 颜色均匀一致，无色差
- [ ] 无透底（面漆盖住了底漆和腻子）
- [ ] 无流挂（漆液流下的痕迹）
- [ ] 无起皮、无裂纹
- [ ] 分色线平直清晰（美纹纸撕得干净）
- [ ] 踢脚线、开关插座边缘收口干净

**摸：**
- [ ] 手感光滑，无颗粒感
- [ ] 墙面不掉粉（用手摸一下，手上没白粉）

**照：**
- [ ] 灯光侧照无波浪纹
- [ ] 无眩光（局部过亮）

#### 阴阳角验收

- [ ] 阳角垂直顺直
- [ ] 阴角整洁干净
- [ ] 护角条无外露

### 🚨 常见验收问题

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 灯光下明显波浪纹 | 🔴 严重 | 重新批腻子，打磨平整 |
| 墙面起皮脱落 | 🔴 严重 | 铲除问题区域，重新刮腻子刷漆 |
| 颜色不均匀 | 🟡 中等 | 补刷一遍面漆 |
| 分色线毛糙 | 🟡 中等 | 修补分色线 |
| 少量针孔/小气泡 | 🟢 轻微 | 点补即可 |
| 开关插座边缘不齐 | 🟢 轻微 | 补漆收口 |

### 🔧 后期修补

刷完漆后，后续安装（开关面板、灯具、柜体）免不了磕碰墙面，**留一瓶同色号的漆**用于补漆。漆的保质期约2年，密封好放在阴凉处。
