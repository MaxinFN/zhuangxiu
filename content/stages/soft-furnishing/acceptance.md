## 软装验收标准

软装验收的重点是**家具牢固性、功能完好、无运输损伤、摆放尺寸合理**。大多数软装产品已在工厂质检，现场重点在检查运输和安装损伤。

---

## 一、沙发验收

- [ ] 框架牢固：两人同时坐下+起身，沙发无异响、无晃动
- [ ] 坐垫回弹：坐下5分钟后起身，坐垫在1-2分钟内恢复原形（高密度海绵≥35D的标志）
- [ ] 面料：无破损、无污渍、无色差、无线头
- [ ] 沙发脚：所有脚平稳着地（推四角无翘脚），可微调脚高
- [ ] 科技布/真皮沙发：表面无划痕
- [ ] 可拆洗款：拉链完好、拆装顺畅
- [ ] 电动功能位（如有）：伸展/复位顺畅、电机噪音≤55dB

## 二、床与床垫验收

### 2.1 床架

- [ ] 床架稳固：推床头板无晃动、无嘎吱声
- [ ] 排骨架：每根排骨条平稳在支撑条上，间距≤50mm
- [ ] 所有螺栓/偏心轮已拧紧到位
- [ ] 木材表面无划痕、无磕碰、无开裂、无色差

### 2.2 床垫

- [ ] 表面无破损、无污渍、无线头脱出
- [ ] 按压各区域回弹均匀（无局部塌陷或硬块）
- [ ] 床垫平放边缘与床架四边齐平
- [ ] 验收确认后撕掉床垫保护膜（膜不透气，长期不撕内部发霉）

## 三、窗帘验收

- [ ] 罗马杆/轨道安装牢固：向下轻拉中间部位无松动
- [ ] 罗马杆/轨道水平：水平尺≤3mm偏差
- [ ] 窗帘开合顺畅（从左到右无卡阻）
- [ ] 落地帘离地10-20mm均匀一致
- [ ] 褶皱均匀丰满（展开后自然美观）
- [ ] 面料无破损、无污渍、无色差
- [ ] 电动窗帘（如有）：遥控/App控制正常、开合到位、遇阻自停有效

## 四、家电验收

### 4.1 电视机

- [ ] 屏幕无碎裂、无坏点（纯色图检查）
- [ ] 底座/挂架安装牢固
- [ ] 开机正常、系统流畅
- [ ] HDMI/USB端口全部可用

### 4.2 冰箱

- [ ] 外观无划痕、无凹陷
- [ ] 门封条密封良好（A4纸夹测法）
- [ ] 搬运后静置≥4-6h再通电（让压缩机油回流）
- [ ] 冷藏/冷冻室温度正常（开机2h后）
- [ ] 运行声音均匀无异常噪音

### 4.3 空调

- [ ] 室内机水平安装（水平尺偏差≤3mm）
- [ ] 室外机安装牢固、有减震垫
- [ ] 制冷/制热正常、出风口温度明显变化
- [ ] 冷凝水管排水通畅（室外可见水滴滴落）
- [ ] 遥控器功能正常

### 4.4 烟机灶具

- [ ] 烟机水平安装、各档位正常、照明亮
- [ ] 烟管连接至公共烟道或室外
- [ ] 止逆阀方向正确（只出不进）
- [ ] 燃气灶点火正常（每灶眼连续3次成功）
- [ ] **熄火保护有效**：点火后手动吹熄火焰→听到"咔"声→燃气断供
- [ ] 火焰蓝色（充分燃烧）——黄色或红色火焰说明燃烧不充分

---

## 五、摆放尺寸验收

- [ ] 动线通道：主要通道≥800mm、次要通道≥600mm（实测）
- [ ] 沙发与茶几间距400-500mm
- [ ] 床与衣柜间距≥600mm（衣柜门全开仍可通过）
- [ ] 床垫高度（床架+床垫）480-580mm
- [ ] 餐桌上方吊灯距桌面700-800mm

## 常见问题处理

| 问题 | 处理 |
|------|------|
| 家具表面运输划痕 | 轻微—商家补漆修复；严重—要求换货 |
| 沙发坐垫回弹慢或不回弹 | 海绵密度不足→联系商家换货或升级填充 |
| 窗帘离地不均匀 | 调整轨道吊码高度→重新挂帘 |
| 电器外观磕碰 | 签收前检查→当场拒收或联系换货 |
| 燃气灶熄火保护不动作 | 联系安装方或品牌售后（**燃气安全无小事**） |
