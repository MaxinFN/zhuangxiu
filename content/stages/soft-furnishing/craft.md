## 软装布置核心技巧

软装决定了空间的实际居住感受。硬装是骨架，软装是血肉。好的软装布置遵循三大原则：**动线优先、视觉层次、尺寸协调**。

---

## 一、空间动线

### 1.1 通道宽度标准

| 通道类型 | 最小宽度 | 舒适宽度 | 说明 |
|---------|---------|---------|------|
| 主要动线（玄关→客厅→走廊） | 800mm | 1000-1200mm | 两人交错通过不碰肩 |
| 次要动线（床边→衣柜/卫生间） | 600mm | 700-800mm | 单人通过+手提物品 |
| 厨房操作通道（单排柜） | 900mm | 1000-1200mm | 转身+蹲下取物 |
| 厨房操作通道（双排柜） | 1000mm | 1100-1300mm | 两人同时操作 |
| 餐桌椅后方通道 | 600mm | 750-900mm | 人坐下时后方可过人 |
| 床与衣柜之间 | 600mm | 700-800mm | 衣柜门全开+人站立 |

### 1.2 功能区动线

| 功能区 | 动线要点 |
|--------|---------|
| 厨房 | 冰箱→水槽→操作台→灶台→装盘区，形成"拿→洗→切→炒→装"的三角动线 |
| 卫生间 | 洗手台→马桶→淋浴区，三者不交叉（干湿分离原理） |
| 卧室 | 进门→衣帽间/衣柜→床→窗，路径最短 |
| 玄关 | 进门→放钥匙/包→换鞋→挂外套，各步骤触手可及 |

## 二、视觉层次

### 2.1 空间重心

| 空间 | 视觉重心（焦点） | 次重心 |
|------|----------------|--------|
| 客厅 | 电视墙/沙发背景墙/落地窗 | 单人沙发+落地灯阅读角 |
| 餐厅 | 餐桌+吊灯组合 | 餐边柜/装饰画 |
| 主卧 | 床头背景墙+床 | 飘窗/梳妆台 |
| 书房 | 书架墙 | 书桌+台灯 |

> 每个空间 **1-2个视觉焦点** 即可，太多会感到杂乱、焦虑。

### 2.2 色彩3+3法则

- **3个主色**：墙面+地面+大家具（沙发/床/柜体），占空间颜色70-80%
- **最多3个点缀色**：抱枕、挂画、地毯、绿植，占20-30%
- 主色选择建议：同色系（单色方案）最安全；对比色（如蓝墙+橙抱枕）需要一定审美功底
- 黑白灰不算颜色，是"无色调节器"——可以用来缓冲其他颜色

### 2.3 材质层次

一个空间中应有 **至少3种不同材质** 的对话：
- 硬（石材/玻璃/金属）+ 软（布艺/地毯/窗帘）+ 自然（木材/绿植/皮革）
- 全是硬材质→冰冷；全是软材质→无精打采

## 三、家具尺寸与摆放

### 3.1 沙发区

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 沙发长度与墙面比 | 2/3左右 | 说"沙发占墙2/3最协调" |
| 沙发与茶几间距 | 400-500mm | 坐下后膝盖不顶茶几、伸手能拿到茶几上的杯子 |
| 茶几高度 | 比沙发坐垫高0-30mm | 通常380-480mm |
| L型沙发布置 | 贵妃榻不挡主要通道 | — |

### 3.2 餐桌区

| 参数 | 推荐值 |
|------|--------|
| 餐桌宽度 | 800-900mm（4人）/ 900-1000mm（6人）/ 1000-1100mm（8人+） |
| 每人用餐宽度 | 600-700mm（舒适）/ 500mm（紧凑） |
| 餐椅拉出后深度 | 椅子本身深+400-500mm（拉出距离） |
| 吊灯距桌面 | 700-800mm（不挡视线、聚焦食物、照亮而不刺眼） |

### 3.3 卧室区

| 参数 | 推荐值 |
|------|--------|
| 床高度（床架+床垫） | 480-580mm（膝盖上缘高度，方便上下床） |
| 床边通道 | ≥600mm |
| 床头柜高于床垫 | 0-50mm（或同高） |
| 挂画中心高度 | 约1550mm（亚洲成年人视线高度） |
| 电视中心高度（卧室） | 约1100mm（卧姿视线高度，比坐姿低） |

### 3.4 挂画与装饰

| 参数 | 推荐值 |
|------|--------|
| 挂画中心距地（站立观看区，如走廊） | 1550-1600mm |
| 挂画中心距地（坐姿观看区，如沙发上方） | 约1350mm |
| 装饰画间距 | 50-80mm（大幅）/ 30-50mm（小幅组画） |
| 挂画宽度比下方家具 | 2/3到3/4（画太宽压家具，太窄显小气） |
| 沙发上方挂画底部距沙发靠背 | 150-250mm |

## 四、窗帘安装

### 4.1 安装位置

| 参数 | 推荐值 | 视觉效果 |
|------|--------|---------|
| 罗马杆/轨道安装高度 | 窗框上沿+100-200mm | **视觉拉高窗户**，层高显得更高 |
| 罗马杆/轨道超出窗框宽度 | 每侧+150-250mm | **视觉拓宽窗户**，窗帘收起时不会遮挡窗户 |
| 落地帘离地 | 10-20mm | 不拖地（易脏），不留过大空隙（像吊脚裤） |
| 飘窗内窗帘 | 窗框内安装，帘子落在台面上 | — |

### 4.2 窗帘尺寸计算

```
窗帘宽度 = [墙宽（满墙帘）或窗宽+两侧延长量] × 褶皱倍数

褶皱倍数：1.5倍（经济紧凑）/ 1.8-2.0倍（标准美观）/ 2.5倍（豪华丰满）
```

## 五、灯光层次

### 5.1 三层照明法

| 层级 | 作用 | 光源 | 色温建议 |
|------|------|------|---------|
| **基础照明** | 均匀照亮全空间 | 主灯/筒灯/泛光天花灯带 | 客厅4000K中性光或3500K暖白 |
| **功能照明** | 特定活动区域精准照亮 | 阅读灯/书桌台灯/厨房操作台灯带/镜前灯 | 4000K（工作区）/ 3000K（阅读） |
| **氛围照明** | 营造空间层次和情感 | 落地灯/壁灯/展柜灯带/烛台 | 2700-3000K暖光 |

### 5.2 各空间灯光方案

| 空间 | 基础照明 | 功能照明 | 氛围照明 |
|------|---------|---------|---------|
| 客厅 | 筒灯/轨道灯（均匀分布） | 沙发旁阅读落地灯 | 电视墙背景灯带、装饰画射灯 |
| 餐厅 | 餐桌上方吊灯 | — | 餐边柜灯带 |
| 卧室 | 床头壁灯（替代主灯） | 床头阅读灯 | 床底灯带（起夜微光） |
| 厨房 | 集成吊顶平板灯 | **操作台上方灯带**（非暗区切菜） | 吊柜底部灯带 |
| 卫生间 | 集成吊顶灯 | 镜前灯（面部照明无阴影） | — |
| 书房 | 吸顶灯 | 书桌台灯（4000K，Ra≥90高显色） | 书柜内置灯带 |

### 5.3 色温统一原则

> **一个空间内色温必须统一**。切忌客厅一半2700K暖光一半4000K白光——观感混乱。唯一例外：厨房操作台灯带4000K（切菜需清晰）+ 餐厅吊灯2700K（温馨就餐）可分开关控制。

## 六、绿植布置

| 位置 | 推荐植物 | 养护难度 |
|------|---------|---------|
| 客厅落地角落 | 琴叶榕（Ficus lyrata）、龟背竹（Monstera deliciosa）、橡皮树（Ficus elastica） | 中等 |
| 电视柜/边柜 | 虎皮兰（Sansevieria）、椒草（Peperomia）、袖珍椰子 | 低 |
| 窗台/书桌 | 多肉组合、绿萝（Epipremnum aureum）、白掌（Spathiphyllum） | 低 |
| 卫生间（耐阴） | 蕨类（波士顿蕨/铁线蕨）、常春藤 | 低 |
| 悬挂（吊盆） | 绿萝、爱之蔓（Ceropegia woodii）、吊竹梅 | 低 |

> 3-5盆即可。植物的作用是"柔化空间线条、增添自然气息"，过多绿植会显得拥挤且养护负担重。
