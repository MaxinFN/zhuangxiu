## 阶段概述

软装是装修的最后一道大题——家具、窗帘、家电、装饰品陆续进场。硬装决定了家的骨架，软装决定了家的灵魂。这个阶段的重点是**搭配**和**尺寸把控**。

### 本阶段核心工作

1. **家具进场** — 沙发、床、餐桌椅、茶几、电视柜等
2. **窗帘安装** — 布帘+纱帘或百叶窗
3. **家电安装** — 电视、冰箱、洗衣机、空调等
4. **装饰布置** — 挂画、地毯、绿植、摆件
5. **收纳规划** — 衣柜、橱柜内部收纳配置

### ⚠️ 最容易忽略的

> **家具尺寸！** 很多人看图片觉得好看就买了，到家发现太大或太小。买家具前一定要量好空间尺寸，算好过道宽度。

### 软装顺序建议

```
大件家具 → 窗帘 → 家电 → 灯具调整 → 装饰品 → 绿植
先大后小，先实用后装饰。
```
