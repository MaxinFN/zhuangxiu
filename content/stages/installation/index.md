## 阶段概述

安装阶段是装修从"工地"变成"家"的关键转折点。橱柜、门、卫浴、灯具、开关面板等陆续安装到位，家的样子开始显现。这个阶段也是**协调最复杂**的阶段——各厂家安装师傅需要安排好先后顺序。

### 本阶段核心工作

1. **橱柜安装** — 柜体+台面（台面通常分两次：柜体装完→复尺→台面安装）
2. **室内门安装** — 房门、卫生间门、推拉门
3. **卫浴安装** — 马桶、浴室柜、花洒、浴霸
4. **开关面板+灯具** — 全屋面板和主灯、辅灯
5. **厨卫吊顶** — 铝扣板集成吊顶（如果木工阶段没做）
6. **热水器+烟机灶具** — 厨房电器安装
7. **地板安装** — 强化/实木复合地板铺设

### ⚠️ 安装顺序很重要！

```
橱柜柜体 → 烟机灶具（预埋烟管） → 橱柜台面复尺 → 
室内门 → 地板 → 踢脚线 → 
卫浴 → 开关面板 → 灯具 → 
橱柜台面安装 → 厨房水槽+龙头 → 净水器
```

> 地板在门之后装（门套压在地板上收口更好看）；台面在柜体之后（需要等柜体装好再精准测量尺寸）。
