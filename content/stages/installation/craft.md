## 安装工艺要点

安装阶段是将前期采购的成品（橱柜、门、地板、卫浴）精确安装到位的阶段。安装质量有一句行业老话：**七分安装，三分材质**——再好的产品，安装不到位效果大打折扣。

---

## 一、橱柜安装

### 1.1 安装流程

```
现场保护 → 柜体拆包核验 → 柜体组装 → 水平调整 → 墙面固定 → 
吊柜定位安装 → 柜门安装+缝隙调整 → 拉手安装 → 台面复尺 → 
台面安装+接缝打磨 → 水槽/龙头/烟机安装 → 打胶收口 → 验收
```

### 1.2 关键工艺参数

| 工艺节点 | 标准值 | 检验方法 |
|---------|--------|---------|
| 地柜水平度 | ≤2mm/延米 | 激光水平仪+卷尺 |
| 柜体垂直度 | ≤2mm/1m高 | 水平尺贴柜体侧板 |
| 柜门缝隙（上下） | 2-3mm（均匀一致） | 塞尺逐扇检查 |
| 柜门缝隙（左右） | 1.5-2.5mm | 塞尺 |
| 相邻柜门高差 | ≤0.5mm | 手摸、目视 |
| 吊柜安装高度（底边距台面） | 700-750mm（中式）/ 650-700mm（西式岛台） | 卷尺 |
| 吊柜固定 | 每个柜体≥2个M8膨胀螺栓（入墙≥50mm） | — |
| 柜体与墙面缝隙 | 用收口条遮盖，不做打胶处理 | 目视 |

### 1.3 柜门铰链调节

百隆/海蒂诗标准铰链支持三维调节：

| 调节维度 | 螺丝位置 | 可调范围 |
|---------|---------|---------|
| 上下 | 底座固定螺丝 | ±2mm |
| 前后（门重叠量） | 铰链臂上偏心螺丝 | -2mm ~ +4mm |
| 左右（门缝宽度） | 铰杯处调节螺丝 | ±2.5mm |

调节顺序：先调前后（让所有门在同一平面）→ 调上下（对齐门顶底）→ 调左右（让门缝宽窄均匀）。

### 1.4 台面安装

| 材质 | 接缝工艺 | 验收标准 |
|------|---------|---------|
| 石英石 | 专用云石胶+固化剂调色粘接→等固化→400#→800#→1500#水磨抛光 | 接缝平整（指甲划过无凹凸感）、颜色与石面相近 |
| 岩板 | 同色胶粘接→打磨 | 接缝平整 |
| 不锈钢 | 氩弧焊焊接→打磨抛光 | 焊缝平整、无变色 |

- 台面与墙面之间打**中性防霉玻璃胶**（西卡厨卫专用或GE防霉系列），胶条均匀、宽度约3-5mm
- 台面外侧下沿加装**止水线**（防止台面上水流到柜门和地面上）
- 水槽安装方式：**台下盆**（加工费¥200-500元）美观易清洁，台面水可直接抹入水槽；台上盆有凸缘积水垢

## 二、室内门安装

### 2.1 安装流程

```
门套组装 → 门框定位 → 垂直度校准 → 发泡胶填缝 → 固化(≥6h) → 
合页开槽安装 → 挂门扇 → 门扇缝隙调整 → 门锁安装 → 门吸安装 → 打胶收口
```

### 2.2 安装工艺参数

| 工艺节点 | 标准值 | 说明 |
|---------|--------|------|
| 门框垂直度 | ≤2mm/2m | 激光水平仪或磁吸线垂 |
| 门扇与地面缝隙 | 5-8mm（无地暖）/ 8-12mm（有地暖） | 保证通风+地板/地毯厚度空间 |
| 门扇与门套缝隙（侧+上） | 2-3mm/条 | 3条缝隙均匀一致 |
| 合页数量 | 每扇门3个（上/中/下） | 高度>2100mm的门用4个 |
| 合页开槽深度 | 与合页片厚度一致（通常1.2-1.5mm） | 槽太浅合页外凸，门扇有缝 |
| 发泡胶填充 | 门框与墙体空隙全部填满（不偷工） | 隔绝声音和气味 |
| 门锁中心距地 | 1000-1050mm | 手自然握把高度 |
| 门吸位置 | 距门扇开启侧边缘80-100mm | 地吸或墙吸，避免绊脚 |

### 2.3 安装后检验

- 门扇自然悬停在任何位置都保持静止（不会自己飘开或关上）→垂直度合格
- 门锁锁舌与锁扣板对齐顺畅，关门无摩擦声
- 密封条压合均匀（用纸片夹在门扇与门套之间拉动测试，应有明显阻力）

## 三、地板安装

### 3.1 铺装方向

| 原则 | 说明 |
|------|------|
| 顺光铺 | 地板长边与主采光窗方向平行（光影显纹理） |
| 沿长边铺 | 地板长边沿房间长边方向（视觉延伸感） |
| 走廊 | 长边沿走廊方向 |

### 3.2 安装流程

```
地面清理 → 检查地面平整度 → 铺设防潮膜 → 铺设地垫 → 
铺地板（从墙开始逐排铺设） → 留伸缩缝 → 安装踢脚线 → 安装扣条/收边条
```

### 3.3 关键工艺参数

| 工艺节点 | 标准值 | 说明 |
|---------|--------|------|
| 地面平整度 | ≤2mm/2m（强化/多层）/ ≤3mm/2m（实木） | 不达标必须先自流平或找平 |
| 地面含水率 | ≤10%（强化）/ ≤12%（实木复合）/ ≤当地平衡含水率+2%（实木） | 含水率计检测 |
| 防潮膜搭接 | ≥100mm，用防水胶带密封 | 潮气上返导致地板膨胀起拱 |
| 地板与墙面伸缩缝 | 8-12mm | 踢脚线覆盖隐藏 |
| 连续铺设长度 | ≥8m（强化）/ ≥6m（实木/实木复合）须在门口处断开加扣条 | 防止大面积膨胀起拱 |
| 相邻两排板端接缝错开 | ≥300mm | 避免通缝导致薄弱线 |

### 3.4 地暖地面铺装

- 铺装前：地暖开启缓慢升温（每天升≤5°C），运行≥72h排出水泥找平层内的潮气
- 铺装时：地暖温度降至18±3°C
- 铺装后：72h内地暖保持在18±3°C，72h后缓慢升温至正常
- 地暖地板选择：实木复合（多层/三层）/SPC石塑地板，**不能用纯实木**

## 四、卫浴安装

### 4.1 马桶安装

| 步骤 | 工艺要点 |
|------|---------|
| 1. 清洁排污管口 | 管口内外用抹布擦净，检查密封圈槽无杂物 |
| 2. 安装法兰圈（密封圈） | 将法兰圈套入马桶排污口（大头向上）或放入地面排污管内（根据法兰圈类型），圈体居中 |
| 3. 定位放置 | 把马桶对准排污管口缓缓放下，使法兰圈完全入位，用力压紧马桶底座 |
| 4. 安装固定螺栓 | 两侧固定螺栓拧紧（但不过度，防止陶瓷开裂） |
| 5. 连接进水管 | 角阀→编织软管→马桶进水口（连接处放密封垫圈） |
| 6. 打胶密封 | 底座与地面交接处打**中性防霉玻璃胶**（不打水泥——后期维修无法取下马桶） |
| 7. 测试 | 打开角阀进水，满水后冲水3-5次，用纸巾擦拭各接口检查有无渗漏 |

> 安装后 **24h内禁用水冲和坐压马桶**（法兰圈和玻璃胶固化需要时间）。

### 4.2 花洒安装

| 点位 | 参考高度 | 说明 |
|------|---------|------|
| 混水阀中心（手持花洒） | 1100mm | 距地面 |
| 混水阀中心（浴缸用） | 750-800mm | 浴缸上沿上方 |
| 顶喷花洒 | ≥2100mm | 高于家中身高最高者头顶≥100mm |
| 手持花洒插座 | 1200-1400mm | 易于取放 |
| 冷热方向 | **左热右冷**（面向墙面时） | 国际通用标准，接反无法调温（恒温花洒直接无法使用） |

### 4.3 浴室柜安装

- 柜体背板用膨胀螺栓+挂片固定于墙体（禁止仅用自攻螺丝或胶粘）
- 柜体水平度≤2mm（水平尺放在柜体顶板检查）
- 排水管必须安装**存水弯**（P-trap或S-trap），水封高度≥50mm（防止臭气和虫鼠进入）
- 排水软管与墙排/地排口之间用密封圈或玻璃胶密封

### 4.4 浴霸安装

- 风暖浴霸出风口**不能正对淋浴区**（身上有水时风加速蒸发导致体感冷）
- 排气管道必须通向室外或公共排风道（不能排入吊顶内——潮气会使吊顶发霉、龙骨生锈）
- 排气管道使用硬质PVC管（不用铝箔软管——风阻大），长度≤3m
- 电源独立回路（≥2.5mm²），建议从卫生间插座回路取电并经过RCBO漏保

## 五、开关面板与灯具

### 5.1 面板安装

- 面板底座紧贴墙面（墙面不平时垫尼龙垫片，不能用纸片）
- 多联面板（如三开+五孔）面板之间无缝隙
- 水平度检查：目视或水平尺，不能歪斜
- 厨房台面上的带开关五孔插座**开关在上、插座在下**（方便按开关）

### 5.2 灯具安装

| 灯型 | 安装要点 |
|------|---------|
| 吸顶灯 | 用膨胀螺栓固定底座于顶面（不能只用自攻螺丝于石膏板） |
| 吊灯（水晶灯/枝形灯） | 重量>3kg必须用膨胀螺栓或飞机胀栓固定于混凝土/龙骨 |
| 筒灯/射灯 | 开孔前用探墙仪确认孔位无龙骨或线管（筒灯开孔通常Φ75mm） |
| 灯带（LED软灯带） | 粘贴在铝合金灯带槽内（兼具散热和光效）；变压器隐藏于检修口或灯槽内 |
| 镜前灯 | 距浴室柜台面500-600mm，足够照亮面部无阴影 |
| 吊灯（餐桌上方） | 灯底距桌面700-800mm（不遮挡视线、聚焦食物） |
