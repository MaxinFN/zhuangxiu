## 安装工艺要点

### 一、橱柜安装

**流程：**
```
柜体组装 → 水平调整 → 固定墙面 → 柜门安装 → 拉手安装 → 
台面复尺 → 台面安装 → 水槽+龙头安装
```

**关键工艺：**

1. **水平调整**：柜体底部用调节脚找平，保证台面水平
2. **柜体固定**：背板与墙面用膨胀螺栓固定
3. **柜门调节**：铰链可三维调节（上下、左右、前后），保证门缝均匀
4. **台面接缝**：石英石台面用专用胶粘接+打磨，接缝做到几乎看不出来
5. **挡水条**：台面与墙面交接处打防霉玻璃胶

### 二、室内门安装

**流程：**
```
门套组装 → 门框定位 → 发泡胶填缝 → 安装合页 → 挂门扇 → 
安装门锁 → 安装门吸 → 打胶收口
```

**关键工艺：**
- 门框垂直度偏差≤2mm
- 门扇与地面缝隙：5-8mm（保证开关顺畅+通风）
- 门扇与门套缝隙：上面+两侧各3mm
- 合页3个（上中下），承重更均匀
- 门框与墙面缝隙用发泡胶填满

### 三、地板安装

**铺设方向：**
- 顺光铺（沿窗户进来的光线方向，视觉效果最好）
- 沿长边铺（视觉延伸感）

**流程：**
```
地面清洁 → 铺防潮膜 → 铺地垫 → 铺地板 → 安装踢脚线 → 安装扣条
```

**关键工艺：**
- 防潮膜搭接≥100mm
- 地板与墙面留缝8-12mm（伸缩缝，踢脚线遮挡）
- 门口处断开加扣条（避免大面积地板膨胀起拱）
- 有地暖的，铺设前需预热地面，排掉潮气

### 四、卫浴安装

**马桶安装：**
1. 排污管口清理干净
2. 安装法兰圈（密封防臭）
3. 马桶对准放下，用力压紧
4. 打防霉玻璃胶（底座与地面连接处）

> 马桶装好后**24小时内不要使用**，等玻璃胶干透。

**花洒安装：**
- 混水阀中心距地面约1100mm
- 顶喷高度约2100mm（高于家中最高的人）
- 出水口左热右冷

### 五、开关面板+灯具

- 面板水平、紧贴墙面，无歪斜
- 多联面板之间无缝拼接
- 灯具安装时注意底座遮住线孔
- 水晶灯、吊灯必须固定在膨胀螺栓上（不能只靠自攻螺丝）
