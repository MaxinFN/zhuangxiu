## 安装阶段验收

安装验收聚焦每件成品的安装质量：**水平度、缝隙均匀、功能运转正常、接口无渗漏**。"七分安装三分材质"——安装验收的重要性不亚于材料本身。

---

## 一、橱柜验收

| 检查项 | 方法 | 合格标准 |
|--------|------|---------|
| 地柜水平度 | 激光水平仪+卷尺 | ≤2mm/延米 |
| 吊柜水平度 | 同上 | ≤2mm/延米 |
| 柜门缝隙（上下） | 塞尺逐扇 | 2-3mm均匀一致 |
| 柜门缝隙（左右） | 塞尺逐扇 | 1.5-2.5mm均匀一致 |
| 每扇柜门开合 | 手动开关3次 | 顺畅、无异响、门能自停在任意角度 |
| 抽屉抽拉 | 全拉出+全推入 | 顺畅、阻尼缓冲自然、左右不晃动 |
| 拉手 | 目视+手摸 | 安装在同一水平线上、无歪斜、无松动 |
| 台面接缝 | 手指来回划+手电筒侧照 | 平滑无缝感、无凹凸、颜色与台面一致 |
| 台面水平 | 水平尺 | ≤2mm/m |
| 台面后挡水 | 目视+手摸 | 与墙面贴合紧密、打胶均匀（宽3-5mm中性防霉胶） |
| 台下盆安装 | 目视 | 台盆边缘均匀低于台面、接缝密封无渗水 |
| 水槽+龙头 | 开水5分钟，干纸巾检查每个接口 | 纸巾干燥无湿痕 |
| 吊柜固定 | 目视柜体内侧 | 每个吊柜≥2个膨胀螺栓固定于墙体 |

---

## 二、室内门验收

| 检查项 | 方法 | 合格标准 |
|--------|------|---------|
| 门框垂直度 | 激光水平仪或磁吸线垂 | ≤2mm/2m |
| 门扇与地面缝隙 | 塞尺 | 5-8mm（无地暖）/ 8-12mm（有地暖） |
| 门扇与门套缝隙（侧+上） | 塞尺 | 每边2-3mm均匀 |
| **门扇自停测试** | 门开到任意角度松手 | 门扇静止不动——说明门框安装垂直 |
| 门锁 | 开关门锁5次+晃动门扇 | 锁舌对准锁扣板顺畅无卡顿、门扇关闭后不松动 |
| 门吸 | 将门推到门吸位置 | 门吸准确吸住、安装牢固 |
| 密封条 | A4纸夹测 | 门关闭后A4纸夹在门与门套之间抽拉有明显阻力 |
| 漆面 | 目视+手摸 | 无划痕、无磕碰、无手印污渍 |

## 三、地板验收

- [ ] **踩踏无异响**：全屋缓慢走动（不同方向多走几遍），无咯吱声响、无起伏晃动
- [ ] 接缝紧密、无起拱
- [ ] 与墙面伸缩缝8-12mm（踢脚线遮盖）
- [ ] 踢脚线与墙面贴合无缝隙、踢脚线底部紧贴地板
- [ ] 扣条/收边条安装牢固、过渡平滑
- [ ] 表面无划痕、无磕碰

---

## 四、卫浴验收

### 4.1 马桶

- [ ] 底座水平稳固、晃动马桶无松动
- [ ] 冲水≥5次，每次冲净有力，水箱正常补水无长流水
- [ ] 底座周边玻璃胶均匀（中性防霉胶），宽度3-5mm
- [ ] **干纸巾测试**：冲水后用干纸巾擦拭底座四周、进水管接口、水箱底部——纸巾完全干燥无湿痕

### 4.2 花洒

- [ ] 混水阀中心距地约1100mm
- [ ] 左热右冷（亲手触摸确认）
- [ ] 各出水模式切换正常、出水均匀
- [ ] 关水后花洒头无滴漏
- [ ] 各接口干纸巾测试无渗水

### 4.3 浴室柜

- [ ] 柜体安装牢固、水平（水平尺顶板≤2mm）
- [ ] 台盆排水顺畅：满水拔塞→水流快速排净
- [ ] 排水管接口无渗漏（干纸巾反复擦拭）
- [ ] **存水弯**已安装（P-trap，水封高度≥50mm）
- [ ] 镜柜/镜子安装牢固、镜面无划痕

### 4.4 浴霸

- [ ] 取暖/换气/吹风/照明各功能独立测试正常
- [ ] 排气管道为硬质PVC管→通向室外或公共排风道（**非排入吊顶内**）
- [ ] 换气功能开启后排气口有吸风（纸巾测试）

---

## 五、开关面板与灯具

- [ ] 每个面板安装水平、紧贴墙面、无歪斜
- [ ] 多联面板之间无缝隙
- [ ] 每个开关控制对应灯具（逐一测试）
- [ ] 双控开关双向正常
- [ ] 每个插座相位检测仪测试接线正确（左零右火上接地）
- [ ] 灯具安装牢固、正常点亮、无闪烁异响
- [ ] 筒灯/射灯开孔边缘整齐
- [ ] 灯带连续无暗区、变压器隐藏

## 常见验收问题处理

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 马桶底座渗水 | 🔴 严重 | 法兰圈未装到位→拆下马桶→重新安装法兰圈→重新测试 |
| 地板大面积踩踏异响 | 🔴 严重 | 拆除地板→检查地面平整度→重新找平→重新铺设 |
| 门扇自停测试失败 | 🟡 中等 | 门框垂直度不合格→拆除门套→重新安装调垂 |
| 柜门缝隙不均匀 | 🟡 中等 | 调整铰链三维调节螺丝 |
| 排水管接口微渗水 | 🟡 中等 | 拧紧接口或更换密封圈→重新测试 |
| 台面接缝有手感 | 🟡 中等 | 重新打磨抛光接缝 |
| 面板歪斜/玻璃胶断口 | 🟢 轻微 | 调正面板/清除断口重新打胶 |
