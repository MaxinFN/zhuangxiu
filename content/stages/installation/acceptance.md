## 安装验收清单

### 🍳 橱柜验收

- [ ] 柜体水平、垂直（偏差≤2mm）
- [ ] 柜门开合顺畅，无卡顿碰撞
- [ ] 柜门缝隙均匀（≤2mm）
- [ ] 五金（合页、滑轨、拉篮）安装牢固
- [ ] 台面接缝几乎看不到（手指摸无明显凹凸）
- [ ] 台面与墙面打胶饱满均匀
- [ ] 水槽排水顺畅，无漏水
- [ ] 柜体无划痕、无磕碰

### 🚪 室内门验收

- [ ] 门扇开关顺畅，无剐蹭地面
- [ ] 门缝均匀（上+两侧≤3mm，下5-8mm）
- [ ] 门锁开合顺畅，锁舌对孔准确
- [ ] 门吸安装牢固
- [ ] 门套与墙面缝隙打胶收口
- [ ] 门扇表面无划痕、无磕碰

### 🚿 卫浴验收

- [ ] 马桶稳固，无晃动
- [ ] 冲水顺畅，水箱不漏水
- [ ] 底座打胶完整、均匀
- [ ] 浴室柜水平安装
- [ ] 龙头/花洒无渗漏
- [ ] 热水器点火正常，水温调节正常
- [ ] 排水顺畅

### 🛏️ 地板验收

- [ ] 地板平整，踩踏无响声
- [ ] 接缝紧密，高低差≤0.5mm
- [ ] 踢脚线安装平整，与墙面贴合
- [ ] 伸缩缝已被踢脚线遮挡
- [ ] 扣条安装牢固
- [ ] 地板表面无划痕、无磕碰

### 💡 灯具+开关验收

- [ ] 所有开关控制正常
- [ ] 所有插座通电（用检测仪逐个测试）
- [ ] 开关面板水平、贴合墙面
- [ ] 双控开关正常工作
- [ ] 漏电保护器正常工作
- [ ] 灯具无闪烁、无电流声

### 🚨 常见验收问题

| 问题 | 严重程度 | 处理 |
|------|---------|------|
| 橱柜门关不拢/碰一起 | 🟡 中等 | 调节铰链 |
| 地板踩踏有响声 | 🟡 中等 | 局部重铺或消声处理 |
| 马桶底座漏水 | 🔴 严重 | 重新安装法兰圈 |
| 开关面板歪斜 | 🟢 轻微 | 拆下重装调平 |
| 室内门剐蹭地面 | 🟡 中等 | 调整合页或刨门底边 |
