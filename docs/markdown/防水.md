该视频围绕家庭装修防水展开，核心观点是“选对标准比噱头更重要”，结合施工细节和避坑指南，帮你快速掌握防水关键。

一、核心认知

防水必要性

厨卫防水是国家标准强制要求（《住宅室内防水工程技术规范》），属于装修红线，漏做可能引发邻里纠纷、财产损失及健康问题。
填缝剂无法替代防水层：刚性材料易因建筑沉降开裂，需搭配 “刚柔并济” 的专业防水材料。


选材料看标准

市面 “K11、超柔高弹” 等名称多为营销，真正判断好坏需看执行标准（如 GB/T23445-2009、JC/T984-2011 等）。

二、关键标准与场景匹配
标准编号材料类型适用部位核心特点GB/T23445-2009聚合物水泥防水涂料地面（I 型）、墙面（II/III 型）I 型柔韧性强（断裂伸长率≥200%），适合地面抗变形；II/III 型粘接力高（≥1.0MPa），适合墙面防掉砖JC/T984-2011聚合物水泥防水砂浆墙面粘接强度高，适配厚贴 / 薄贴瓷砖，表面拉毛设计提升贴砖牢固性JC/T2415-2017瓷砖胶下防水涂膜瓷砖胶薄贴场景桥接裂缝能力强，但粘接强度仅 0.5MPa，不适合大砖薄贴GB18445-2025水泥基渗透结晶型防水地下室 / 外墙等混凝土结构渗透结晶自愈，适合背水面堵漏，施工成本高，不推荐厨卫

三、施工细节与避坑

部位差异化处理

卫生间地面：用 GB/T23445 I 型柔性涂料，防水层上返 15cm 形成 “盆状” 结构。
卫生间墙面：优先 JC/T984-2011 防水砂浆（或 GB/T23445 II/III 型），淋浴区刷至 1.8m，非淋浴区 30cm。
厨房 / 阳台：地面全柔性（GB/T23445 II 型），阳台需做明显排水坡度，避免积水。


施工流程与时间

刷完一遍需等表干再刷第二遍，整体实干 24 小时后做 48 小时闭水试验（蓄水 20-30cm，观察楼下天花板及管道根部无渗漏即为合格）。


禁用材料

溶剂型涂料（含甲苯等 VOC，有毒易燃）、双组份聚氨酯（不适合通风差的卫生间）、防水卷材（厨卫节点多，易留隐患）。




四、总结建议

优先选 “刚性墙面 + 柔性地面” 组合，根据瓷砖铺贴方式（厚贴 / 薄贴）匹配标准。
预算有限可考虑 JC/T2090 刚性浆料（仅墙面），但需额外找平 / 拉毛，施工较繁琐。
自己施工需严格按产品说明操作，关键节点（阴阳角、管根）多刷几遍，避免凭感觉。

（注：广州地区湿度较高，建议选择耐候性好的柔性涂料，并确保闭水试验彻底。）
