该视频围绕 “瓷砖选购避坑”，从国家标准、性能参数到线下验货技巧，系统梳理了核心要点，可总结为以下 5 部分：
一、质量与环保：选砖底线

关键标准：GB/T4100-2015（物理性能强制标准，需查看检测报告）、GB6566-2010（放射性核素限量，关系家人健康）、GB/T45817-2025（质量分级标准，优先选 5A 等级）；瓷质砖需有 CCC 认证标志。
核心参数：吸水率≤3%（瓷质砖，BⅠA 类），耐磨等级≥4 级（家用高频率空间），断裂模数≥45MPa（不易崩瓷），耐污染性 / 耐化学性达 5 级 / A 级（易清洁），地砖摩擦系数≥0.7（防滑）。

二、吸水率与砖类选择：按空间匹配

瓷质砖（全瓷砖）：吸水率≤3%，1200℃高温烧制，硬度高、不易开裂，规格可选 400×800 至 750×1500，需用专用瓷砖胶铺贴，适合厨房、卫生间等湿区及客厅等高频使用空间。
陶质砖（泡水瓷片）：吸水率 6%-10%，800℃烧制，价格低、施工简单（可水泥砂浆铺贴），但硬度低、易开裂，规格≤400×800，适合预算有限的厨卫墙面。
细炻砖（半瓷砖）：吸水率 3%-6%，施工需泡水且不能用水泥砂浆，又浪费瓷砖胶，规格≤800×800，限制多，不推荐购买。

三、釉面类型：按需求选工艺

抛光砖：镜面效果，表面有细微气孔，易渗污，不建议厨房、卫生间使用。
亮光砖：玻化砖胚体淋釉烧制，防污耐磨优于抛光砖，需选 “淋釉满烧” 工艺，适合客厅干区。
玻化砖：吸水率≤0.5%，质地坚硬、抗污耐磨，适合客厅、卧室等地面。
哑光砖：釉料加消光粉，柔和不刺眼，防滑性好（遇水油不易打滑），适合有老人小孩的家庭及卫生间湿区，选 “多层釉 + 哑光釉” 更易清洁。

四、线下验货：5 步实操技巧

看：表面无气泡、划痕，色泽均匀（自然光下对比同批次砖，避免色差）。
听：轻敲声音清脆，说明磁化程度高、质量好。
刮：硬币 / 钥匙划表面，无痕迹则耐磨。
滑：刀片划砖边缘，不掉粉、无深痕则质地紧密。
滴：背面滴水不渗透（水珠鼓起来），说明密度高、抗污防潮；油性笔写字等几分钟后易擦掉，说明抗污性好。

五、尺寸与色差：避免铺贴隐患

尺寸检测：随机选 4 块砖，测量对角线长度一致（确保直角），边对边靠齐检查厚度均匀、无翘曲，避免铺贴后高低差或缝隙不均。
色差控制：优先选同一批次砖，自然光下平放对比，无明显深浅不一再购买。

总结：选砖时优先 5A 等级、低吸水率（≤3%）、4 级以上耐磨的瓷质砖；湿区选哑光或防滑砖，干区可选亮光砖；线下用 “看、听、刮、滑、滴”5 步快速验货，同时检查尺寸和色差，可大幅降低空鼓、脱落、渗污等风险。
